    老夫长似春三月，游戏人间不皱眉。

曹勋　三月二首
    游子从来悲故乡，发来襟袖芷兰芳。
    梦魂飞去云涛远，杖策闲行秋水光。
    可是梦中曾到此，安知身后却余香。
    紫阳弟子有贤子，卜筑新亭真肯堂。

曹勋　三月十日
    才看桃李锦成围，忽使园林绿作堆。
    远草将人双眼去，飞花引蝶过墙来。
    簿书节里无多著，怀抱朝来得好开。
    已是七分春去了，何须鸟语苦相催。

曹勋　山居杂诗九十首
    种杏三年实，种桃一栽花。
    花实固可乐，为农本桑麻。
    功成不占地，改植常可夸。
    要是农须学，作略自一家。

曹勋　山居杂诗九十首
    我生不侥福，始终躬纯臣，
    虽更齐楚伪，常障西风尘。
    耳顺早引年，挂冠逾八春。
    甘与田畯游，遂予胸次真。

曹勋　山居杂诗九十首
    农人作田务，耘者最辛苦。
    肘膝伏泥涂，拔莠连茹取。
    所拔随已多，悉酿所伏处。
    惟冀禾稻肥，岂问正炎暑。

曹勋　山居杂诗九十首
    隔岁种成麦，起麦秧稻田。
    晚禾亦云竟，冬菜碧相连。
    收割不闲手，垅亩无空阡。
    家蓄千指客，始知常有年。

曹勋　山居杂诗九十首
    孟夏物物茂，瓜瓠行置架。
    引苗上扶疏，须蔓竞缠挂。
    白花亦已出，青实旋造化。
    会喜园枯时，匕箸不增价。

曹勋　山居杂诗九十首
    地水与火风，四大各一性。
    和合乃养形，损增则成病。
    圆觉著于品，岐伯峻其证。
    君子知其言，穷理宜至命。

曹勋　山居杂诗九十首
    蒸沙以为城，冶铁用作限。
    未伸彼利害，先为己忧患。
    家事席未暖，基址已永叹。
    在德不在险，推类仁者谏。

曹勋　山居杂诗九十首
    山中四序景，巧供晨夕间。
    茶瓯不少辍，诗魔常畏闲。
    生计付风雨，年华寄毛颜。
    支筇正独立，栖禽相与还。

曹勋　山居杂诗九十首
    吾事今七十，非杖事能行。
    食罢散步武，欲与跛鳖争。
    皮里气不充，骨中髓不生。
    安能勉筋力，笑傲云霞轻。

曹勋　山居杂诗九十首
    归为田舍翁，不作无复理。
    耕凿顷亩地，远近先视水。
    沟塍欲利达，农具无委靡。
    高田下田事，得马失马尔。

曹勋　山居杂诗九十首
    伏日更酷暑，当午剧炎烈。
    火云凝不动，势欲终岁热。
    天公苏赤子，雨随震霆发。
    一扫回清凉，庭筠助骚屑。

曹勋　山居杂诗九十首
    小圃艺花竹，春末已繁秀。
    土脉得锄斸，枝叶易秀茂。
    久以告老病，未免闭圭窦。
    相忘无芥蒂，一二约亲旧。

曹勋　山居杂诗九十首
    除地台山麓，稍葺五亩宅。
    松竹互偃亚，云山若阡陌。
    子舍念我老，文余躬干役。
    欣欣归来乐，乐岂一朝夕。

曹勋　山居杂诗九十首
    古半喜妙供，溪荪种匀抽。
    蘸水浮翠蝟，络石骈栗皱。
    薿薿巧凝绿，梢梢眇含秋。
    相对几案间，萧然无与俦。

曹勋　山居杂诗九十首
    墙里种桃杏，新春亦已花。
    墙外绕扶疏，绿荫皆桑麻。
    今年风雨时，日见田稻嘉。
    长瓶泻黄汤，且饷田畯茶。

曹勋　山居杂诗九十首
    闲里心情拙，还知兴自嘉。
    饥鸦时立槛，野蝶自寻花。
    客至聊从容，僧来不厌茶。
    晚烟时伫立，列岫映余霞。

曹勋　山居杂诗九十首
    清晓看田去，乘露衣襟凉。
    已有耘夫出，揖我喜非常。
    为言苗正好，喜颜笑亦长。
    虽有吏在门，亲苗端可偿。

曹勋　山居杂诗九十首
    人生行乐尔，行乐则心闲。
    尽既无所用，和气滋容颜。
    卫生有此妙，余事皆可删。
    若益兼忘理，敢告知识间。

曹勋　山居杂诗九十首
    松根有至药，琥珀与茯苓。
    琥珀未易致，茯苓选宜精。
    制与鞠尘细，色如截肪并。
    一一伴朝粥，地黄安可京。

曹勋　山居杂诗九十首
    招客图客欢，不图能病客。
    既醉失匕箸，吐茵堕巾帻。
    何堪款曲意，贻念风月夕。
    拂旦往问讯，惭恨见颜色。


曹勋　山居杂诗九十首
    松色荫人居，松力治人病。
    丹砂含其滋，复何延性命。
    幽居绕苍髯，不畏夏日盛。
    风来听笙竽，便有华胥兴。

曹勋　山居杂诗九十首
    春雪翳重霄，寒雨连日夜。
    不辍兽炭炽，甚恐庭花谢。
    疏帘莫御冷，炉焚屡成灺。
    寒食数日间，春事求诸野。

曹勋　山居杂诗九十首
    吾年逾七十，真欲原春迟。
    岂谓上元过，适丁多雨时。
    韶华荏苒半，清明旦夕期。
    流莺与蜂蝶。冷落殊未知。

曹勋　山居杂诗九十首
    卜居得幽胜，景物无升沉。
    峰峦环郛郭，松竹排园林。
    时时破寂寥，悠悠赖鸣禽。
    於焉自有得，天籁皆余音。

曹勋　山居杂诗九十首
    客寄辇毂下，驰心田亩间。
    君恩重山岳，臣节殊安闲。
    虽幸米盐足，其如尘土颜。
    终负告归罪，判然诚独难。

曹勋　山居杂诗九十首
    朼去天台居，邑有万户酒。
    一出尝十家，已觉醉屡肘。
    留连多好事，方圆烦饤斗。
    约趁黄昏归，不知月挂柳。

曹勋　山居杂诗九十首
    年来心目昏，稍息文字业。
    晓拥台山云，时读修禊帖。
    稚子来捋须，幼孙每持镊。
    晚来微雨休，幽径无不涉。

曹勋　山居杂诗九十首
    老去情怀懒，修然只此翁。
    闲窗千里月，修竹一帘风。
    事与孤鸿远，心将止水同。
    故人如问讯，痴坐正观空。

曹勋　山居杂诗九十首
    寂历秋风晚，闲庭落叶深。
    天高晴日澹，山净暮烟沉。
    疾病无时已，年华有底侵。
    两行南去雁，一片北归心。

曹勋　山居杂诗九十首
    三伏暑可畏，贫富方共之。
    雨忽兼飘风，凉与清气随。
    一饷涤炎暍，万里失蒸炊。
    大哉天地德，好生常所施。

曹勋　山居杂诗九十首
    北归了无日，先垄扫何时。
    愁与燕鸿远，心同楚客悲。
    寒蝉咽败柳，幽鹭来清池。
    谁与话旧事，泪藓空在衣。

曹勋　山居杂诗九十首
    七月正流火，晚饭净刹中。
    稍脱蒸郁苦，顿觉心境空。
    放意览翠岫，徐行迎好风。
    得来喜不彻，此赐皆天公。

曹勋　山居杂诗九十首
    连日山村雨，牛羊都在栏。
    低田渍积水，卧穗铺新寒。
    气逼琴丝润，香传岩桂丹。
    天光惜垅亩，应放月团团。

曹勋　山居杂诗九十首
    弃官了无事，饭余心境空。
    清琴或数弄，浊酒时一中。
    人生苦不闲，得闲须从容。
    闭门适吾乐，三籁皆同功。

曹勋　山居杂诗九十首
    秋色遂将半，游云时淡浓。
    水光影{禾罢}稏，秀色增芙蓉。
    斸地种三径，钩窗来五峰。
    新晴何以卜，一意听晨钟。

曹勋　山居杂诗九十首
    今年已断梅，七月雨益大。
    苗虽不厌雨，灌恐有余溉。
    禾梗正柔嫩，水满舞澎湃。
    其间田高低，于焉有畏爱。

曹勋　山居杂诗九十首
    北风忽卷热，山顶云已驰。
    日影未容敛，雨脚俄骤垂。
    一向得凉吹，四山横修眉。
    暄寒本两陆，莫能测密移。

曹勋　山居杂诗九十首
    冷趁西风便，云趋南国深。
    相舂闻野唱，漂絮击溪砧。
    小立迟残照，闲愁倦寸心。
    新亭有耆旧，应共惜花阴。

曹勋　山居杂诗九十首
    白日忽若暮，北风天正寒。
    浓阴欲酿雪，云侣商量间。
    群儿竞喜跃，婢子争仰看。
    命席拨瓮碧，取醉期尽欢。

曹勋　山居杂诗九十首
    今岁立秋早，便觉驱探汤。
    虽有正午热，已觉中夜凉。
    麻豆率房角，早禾亦上场。
    吾心喜可知，纪实藏诗章。

曹勋　山居杂诗九十首
    晚家台山下，云水同一乡。
    人事讲邻曲，鸡黍期相当。
    秋风起木杪，凉气吹衣裳。
    东家喜见招，社瓮开新尝。

曹勋　山居杂诗九十首
    凄风入林薄，昪日升山迟。
    槿花落因雨，菊英开映篱。
    身老每倦客，目昏犹喜诗。
    所欣水云念，得赴猿鹤期。

曹勋　山居杂诗九十首
    道人谈长生，禅子论寂灭。
    二子互出奇，生灭孰判别。
    吾侪守黄卷，生理知犹劣。
    惟当一以贯，无问亦无说。

曹勋　山居杂诗九十首
    晨起冒清露，笋　　出　　闉。
    原野回旧色，花柳摇新春。
    眷言山林性，常与鱼鸟邻。
    恰然适襟抱，自谓羲皇民。

曹勋　山居杂诗九十首
    我本汝颍士，老隐天台山。
    一瓢顾粗足，三谒无厚颜。
    身与樵牧远，居眇林壑间。
    不为物事役，常对风日闲。

曹勋　山居杂诗九十首
    前岩有兰若，委径宜细履。
    未能望窣堵，屡复涉清泚。
    路纡疲脚力，僧老识茶味。
    策杖趁栖鸦，斜阳在林际。

曹勋　山居杂诗九十首
    客惠双井芽，千里冒寒暍。
    品格自高胜，香味两奇绝。
    小小猷溪磨，团团掌中月。
    挥肘喜旋之，纷泊看落雪。
