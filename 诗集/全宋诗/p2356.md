    看来看到无看处，光境俱忘一卷终。

释惟一　风幡亮上人游浙
    动在风幡动在心，集云棒下已分明。
    而今百越三吴去，肯听它家热碗鸣。

释惟一　古堂
    过了威音佛已前，向虚空里架榱椽。
    至今景定年之二，往却门风尚俨然。

释惟一　观音赞
    耳门圆照，日月两间。
    悲原洪深，衣被四海。
    方便智慧，靡不圆融。
    见闻觉知，无能障碍。
    人间天上独优游，是则名为观自在。

释惟一　荷露跳珠
    优昙初现叶团团，错落明玑走碧盘。
    沆瀣莫将鱼目比，看时容易觅时难。

释惟一　黄蘖断际禅师赞
    大机磅礴，大用纵横，
    要掌便掌。大中天子，
    掌中突出，要棒便棒。
    济北大树，棒头挺生。
    大唐国里只一人，千古仰之日月明。

释惟一　吉州圆上人之浙
    庐陵米价没譫讹，半说低兮半说高。
    此话要圆缄却口，听他江浙闹嘈嘈。

释惟一　偈颂四首
    腊月尽是岁除夜，明朝又是正月一。
    年去年来无尽期，拄杖依前乌律律。

释惟一　偈颂四首
    叔季精神元似旧，慧慈面目只如新。
    来年年是去年年，今日日同明日日。

释惟一　偈颂四首
    今岁今宵尽，日日日从东畔出。
    明年明日来，朝朝鸡向五更啼。

释惟一　偈颂四首
    腊月二十八九，家家户户尽有。
    不是北禅露地牛，便是清源白家酒。
    罗旬杯盘，同欢共笑。

释惟一　偈颂一百三十六首
    是法住法位，世间相常住。
    佛殿对三门，灯笼挂露柱。

释惟一　偈颂一百三十六首
    浴冰桥畔，伏虎岩前。
    雪消松杪，春到梅边。
    古佛眼睛突出，瞿昙面目昭然。
    认著依前隔大千。

释惟一　偈颂一百三十六首
    夜夜明星现，有眼谁不见。
    此夜老瞿昙，却被明星转。
    直至而今双眼花，无瑕白玉自生瑕。

释惟一　偈颂一百三十六首
    和暖上元天气，是处笙歌动地。
    瑞岩无可设施，随分放灯一二。

释惟一　偈颂一百三十六首
    今朝十五，明朝十六。
    瓠苦瓜甜，松直棘曲。
    辟支老汉，炎夏衣楮，
    严冬浴冰，谩自辛勤，
    徒劳检束，争似而今禅和，
    三三两两，水边林下，
    寒向火，热乘凉，
    又谁管你地覆天翻，天翻地覆。
    是玉也大奇，只恐不是玉。

释惟一　偈颂一百三十六首
    露冷风清，山空月明。
    天际归鸿书梵字，夜深砧杵送秋声。
    眼觑不见，耳听不闻。
    与君倾尽此时心，莫认山河作眼睛。

释惟一　偈颂一百三十六首
    伏虎岩前，螺江江上。
    一去一来，无去来相。
    今朝辟支，昔日翁和尚。

释惟一　偈颂一百三十六首
    微雨润枯莇，严风搅林木。
    梅唇已破香，柳眼眠未足。
    尘世竞奔忙，光景相追逐。
    林间无事人，好唱云门曲。
    今朝腊月二十五，明朝腊月二十六。

释惟一　偈颂一百三十六首
    今日上元令节，雨意连绵未歇。
    过去燃灯如来，全体一时漏泄。
    檐头滴滴，分明历历。

释惟一　偈颂一百三十六首
    建岭云，建溪水。
    缥缈无边，渊深无底。
    尽情收拾归来，点滴无非珍贵。
    抛向诸人面前，切忌自生拟议。
    见之不取，思之千里。

释惟一　偈颂一百三十六首
    禅禅，风生木杪，
    叶坠庭前。唧唧寒蛩吟破壁，
    声声归雁度辽天。诸人若也会得，
    此去吴屯不远。若也不会，
    蓝溪水透洋边。阿呵呵，
    笑倒武夷张大仟。

释惟一　偈颂一百三十六首
    皎洁中秋月，清光处处同。
    是人皆照破，无水不形容。
    圆满纤尘净，婆娑桂影重。
    古今争赏玩，谁到广寒宫。

释惟一　偈颂一百三十六首
    旧日重阳，今日重阳，
    篱菊披金浅浅黄。休讶去年蓬鬓，
    今岁半成霜。喜得同人相访，
    那边风景，此间风景，
    且莫论量。相与对花赊一醉，
    高歌大笑，手舞足蹈，
    听教人道掣颠狂。

释惟一　偈颂一百三十六首
    虎溪冰莹长空月，凤岭松吟后夜风。
    堪笑堪悲翁老子，俨然不改旧时容。

释惟一　偈颂一百三十六首
    可祖齐腰立雪，长庆坐破蒲团。
    一人铁壁觑要透，一人沧海沥须乾。
    铁壁透，沧海乾，
    出礼三拜依位立，卷起帘来天下宽。
    堪悲堪笑，何易何难。
    信道梅花香扑鼻，须是彻骨一番寒。

释惟一　偈颂一百三十六首
    罗绮丛中，笙歌社里。
    灯明如来，放光现瑞。

释惟一　偈颂一百三十六首
    秋风卷地，秋水连天。
    千山影瘦，万木萧然。
    渔笛数声江上月，樵歌一曲岭头烟。

释惟一　偈颂一百三十六首
    杨岐三脚驴，虎丘睡虎子。
    毛色既一同，蹄爪亦相似。

释惟一　偈颂一百三十六首
    无碍慧门，铁壁铁壁。
    一击击开，不劳余力。
    几人把手拽不入。

释惟一　偈颂一百三十六首
    横一丈，竖一丈。
    满室风生，半窗月上。
    山僧要坐便坐，要起便起，
    终不向这里无风起浪。诸人若不散去，
    照雇劈脊拄杖。

释惟一　偈颂一百三十六首
    目视云霄，有一拳未敢分付。
    日游闹市，三十棒且待别时。
    遇坎则止，乘流而行。
    既无迹於去来，宁有心於动静。
    昨日恁么，逸翮独翔，
    孤风绝侣。今日不恁么，
    玩江月，听松风。
    拈起合皂山，浓蘸清江水。
    平展金凤洲，写出祖师意。
    贴向十字街头，俾人人知道，
    一年十二月，腊月是年尾。
    宝佑改元，又从头起。
    万年万年更万年，清江依旧清无底。
    道泰不传天子令，时清休唱太平歌。

释惟一　偈颂一百三十六首
    不如随分纳些些，何处青山不是家。
    玉笛横吹皇化里，风前清韵彻天涯。

释惟一　偈颂一百三十六首
    惠力不宝藏，日转万亿匝。
    一万意亿匝，千日千万亿。
    原以转轮数，仰祝天子寿。
    惟原天子寿，复倍转轮数。
    繁开玉叶茂金枝，永与法轮无尽时。

释惟一　偈颂一百三十六首
    以柳堤边舞，黄莺树上鸣。
    瞿昙真实相，历历在而今。
    谁言入灭向双林。

释惟一　偈颂一百三十六首
    这个老汉，殊无所解。
    不惟屈屈曲曲，且是顽顽赖赖。
    我昔没兴见之，直到而今尀尀。
    烧炷香，礼三拜吃拳打拳，
    彼此莫怪。

释惟一　偈颂一百三十六首
    今朝结制，号令宜先。
    三章约法，普示人天。
    身不可妄动，亦不可块然。
    口不可妄开，亦不可默然。
    意不可妄举，亦不可罔然。
    依如是行如是住，圆觉伽蓝常现前。

释惟一　偈颂一百三十六首
    狮子吼，无畏说，
    百兽闻之皆脑裂。

释惟一　偈颂一百三十六首
    昼也惺惺，夜也惺惺。
    既不掉举，亦不昏沉。
    何须求白玉，只此是黄金。

释惟一　偈颂一百三十六首
    夏热乘凉，冬寒向火。
    自古自今，总恁么过。
    不知二祖立雪齐腰，毕竟图个什么。
    以吾之不可，学柳下惠之可。

释惟一　偈颂一百三十六首
    无禅不参，无道可学。
    只责诸人如醉而醒，如梦而觉。
    醉既醒，梦既觉。
    虽然只是旧时人，却与旧进相辽邈。

释惟一　偈颂一百三十六首
    德山拆殿，宝峰烧香。
    二俱不恶，一种不臧。
    瞿昙元自面皮黄。

释惟一　偈颂一百三十六首
    开门待知识，知识喜来过，
    两两不成双，一一无回互。
    共卧不共床，同行不同步。
    昭然在目前，目前不可睹。
    虽则不可睹，我独知其故。
    圣容菩萨来西天，庐仙夏人在东土。

释惟一　偈颂一百三十六首
    山前一片闲田地，无意求之却宛然。
    几许风光都买尽，不曾费著一文钱。

释惟一　偈颂一百三十六首
    今朝解制，且作么生。
    从前约束，一概放行。
    万里秋天飞一鹗，东西南北任纵横。

释惟一　偈颂一百三十六首
    昨人天上去，元不离地下。
    今归地下来，亦不离天上。
    上下无间，来去无差。
    天河秋一雁，砧杵夜千家。

释惟一　偈颂一百三十六首
    六年雪岭竟无成，练得身形似鹤形。
    尽谓见明星悟道，何曾悟道见明星。
    入山何富，出山何贫，
    知心能几人。

释惟一　偈颂一百三十六首
    春风料峭，春雨连绵。
    桃花逐浪，新竹笼烟。
    香严已往严云去，一度春深一怆然。

释惟一　偈颂一百三十六首
    这老冻侬，惺惺懵懂。
    分明无一丝毫，刚道有三斤重。
    只知恁么脱空相诳，不知其雠甚於戴天之不共。

释惟一　偈颂一百三十六首
    去从闹市红尘去，归向白云深处归。
    因甚更无迎与送，去来寸步不曾称。

释惟一　偈颂一百三十六首
    细不可减，大无以加。
    近在眼睫，远过天涯。
    是著即错，拟之又差。
    缺齿老胡会不得，空携只履返流沙。

释惟一　偈颂一百三十六首
    五五午时书，妖邪尽扫除。
    一堂风悄悄，千古意如如。
