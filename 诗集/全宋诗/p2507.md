    恁麽则不去也，此语已行遍天下，
    摘杨华，摘杨华，

释宗杲　颂古一百二十一首
    涂路波吒数十州，传言送语当风流。
    不知脚下泥生刺，蹋著锥人脚指头。

释宗杲　颂古一百二十一首
    直饶钉得遮一缝，点检将来非好手。
    可怜两个老禅翁，却向俗人说家丑。

释宗杲　颂古一百二十一首
    鉴坏十方常住地，三钱使尽露尸骸。
    罗山古佛虽灵验，未免将身一处理。

释宗杲　颂古一百二十一首
    同门握手话分携，古路迢迢去莫追。
    却笑波心遗剑者，区区空记刻舟时。

释宗杲　颂古一百二十一首
    一挝涂毒闻皆丧。身在其中总不知。
    八十翁翁主场屋，真诚不是小儿嬉。

释宗杲　颂古一百二十一首
    钟声披起郁多罗，碧眼胡儿不奈何。
    一箭双雕随手落，拈来元是栅中鹅。

释宗杲　颂古一百二十一首
    入门便棒，郎当不少。
    依而行这，胡麻厮缴。

释宗杲　颂古一百二十一首
    不落不昧，石头土块。
    蓦路相逢，银山粉碎。
    拍手呵呵笑一场，明州有个憨布袋。

释宗杲　颂古一百二十一首
    俊哉一跃透重渊，霹雳迫之去不还。
    却笑龙门烧尾者，依前点额在波澜。

释宗杲　颂古一百二十一首
    南泉打破闲家具，浩浩诸方作话看。
    今日为君重举过，明明历历不顢頇。

释宗杲　颂古一百二十一首
    背触非遮护，明明直举扬。
    吹毛虽不动，遍地是刀枪。

释宗杲　颂古一百二十一首
    瑞岩家风，唤主人公。
    昨夜南山，虎咬大虫。

释宗杲　颂古一百二十一首
    镆铘在握，天魔胆落，
    明眼衲僧，休更卜度。

释宗杲　颂古一百二十一首
    丹山生鹡鷟，师子产狻猊。
    棒下摩醯眼，徒夸第一机。

释宗杲　颂古一百二十一首
    对从全提摩竭令，岂是闲开两片皮。
    喝下瞎驴成队走，梦中推倒五须弥。

释宗杲　颂古一百二十一首
    暗中携手上高山，及至天明各自行。
    无限中涂未归客，明明开眼堕深坑。

释宗杲　颂古一百二十一首
    阳焰何曾能止渴，画饼几时充得饥。
    劝君不用栽荆棘，后代儿孙惹著衣。

释宗杲　颂古一百二十一首
    眼中无翳休挑刮，镜上无尘不用磨。
    信脚出门行大路，横担拄杖唱山歌。

释宗杲　颂古一百二十一首
    好个话端，阿谁解举。
    举得十分，未敢相许。

释宗杲　颂古一百二十一首
    全死中全活，全活中全死。
    一个讶郎当，一个福建子。

释宗杲　颂古一百二十一首
    境话会来虽未是，却问如今作麽生。
    清凉元本鼻头直，夹山依旧两眉横。

释宗杲　颂古一百二十一首
    睦州担板，那容眨眼，
    阔狭短长，不须增减。

释宗杲　颂古一百二十一首
    一气转一大藏教，顿渐偏圆权与实。
    无边妙义炳然彰，元来一字也不识。

释宗杲　颂古一百二十一首
    以平报不平，王法本无亲。
    临济虽明眼，也是黄龙精。

释宗杲　颂古一百二十一首
    先师会里呈真处，临济堂前吃菜时。
    连此三回露拴萦，咄这缘台盘乞儿。

释宗杲　颂古一百二十一首
    夜半明星当午现，愚夫犹待晓鸡鸣。
    可怜自屎不知臭，又欲得新牛似人。

释宗杲　颂古一百二十一首
    提起须弥第一槌，电光石火太迟迟。
    象王行处狐踪绝，师子咆哮百兽危。

释宗杲　颂古一百二十一首
    和尚不赴堂，庄王谢临屈。
    一字入公门，九牛撤不出。

释宗杲　颂古一百二十一首
    祖父田园都卖了，四边界至不曾留。
    奈何犹有中心树，恼乱春风卒未休。

释宗杲　颂古一百二十一首
    新妇骑驴阿家达，步步相随不著鞭。
    归到画堂人不识，从今嫩更出门前。

释宗杲　颂古一百二十一首
    猛焰不容蚊蚋泊，大海那能宿死尸。
    任是三头并六臂，望风无不竖降旗。

释宗杲　颂古一百二十一首
    云门乾屎橛，全超法报化。
    无事出山游，百钱杖头挂。

释宗杲　颂古一百二十一首
    巍巍一座大弥卢，荷负非干气力粗。
    纵使不随言语会，却来当面受糊涂。

释宗杲　颂古一百二十一首
    问佛如何是，宗师即便酬。
    称鎚将锯解，言外度迷流。

释宗杲　颂古一百二十一首
    古庙里头回避得，纸钱堆畔暗嗟吁。
    闲神野鬼皆惊怕，只为渠侬识梵书。

释宗杲　颂古一百二十一首
    望州乌石与僧堂，丛识茫茫不可当。
    提起衲僧拄杖子，五湖四海沸如汤。

释宗杲　颂古一百二十一首
    痴人面前休说梦，生铁团上须寻缝。
    明明说与却佯聋，只管外边闲打共。

释宗杲　颂古一百二十一首
    移身不移步，移步不移身。
    走却金师子，捉得玉麒麟。

释宗杲　颂古一百二十一首
    总道见桃华悟道，此语不知还是无。
    茫茫宇宙人无数，那个男儿是丈夫。

释宗杲　颂古一百二十一首
    打破鬼门关，日轮正当午。
    一箭中红心，大地无寸土。

释宗杲　颂古一百二十一首
    言无展事，语不投机。
    随言者丧，滞句者学。
    逢人不得错举。

释宗杲　颂古一百二十一首
    夺得骊珠即便回，小根魔子尽疑猜。
    拈来抛向洪波里，撒手大家空去来。

释宗杲　颂古一百二十一首
    忽尔出门先见路，才方洗脚便登船。
    神仙秘诀真堪惜，父子虽亲不可传。

释宗杲　颂古一百二十一首
    有礼有乐，有唱有酬。
    人平不语，水平不流。

释宗杲　颂古一百二十一首
    瞎驴一众跳皆惊，正法那堪付与人。
    三要三玄俱丧尽，堂堂摆手出重城。

释宗杲　颂古一百二十一首
    撦破云门一柄扇，拗折乾峰一条棒。
    二三千处管弦楼，四五百条华柳巷。

释宗杲　颂古一百二十一首
    见色明心事已差，闻声悟道更交加。
    观音妙智慈悲力，荆棘林生优钵华。

释宗杲　颂古一百二十一首
    剔开金殿锁，撞动玉楼钟。
    泣露千般草，吟风一样松。

释宗杲　颂古一百二十一首
    天地同根伸一问，未曾抬步已亡家。
    无阴阳处华重发，玉本无瑕却有瑕。

释宗杲　颂古一百二十一首
    玄沙三种病人话，透出云门六不收。
    莫待是非来人耳，从前知己返为雠。

释宗杲　颂古一百二十一首
    夫子不识字，达磨不会禅。
    玄沙无此语，切莫妄流传。

释宗杲　颂古一百二十一首
    赤肉团边用得亲，主宾有理各难伸。
    两个驼子相逢著，世上如今无直人。

释宗杲　颂古一百二十一首
    有时笑兮有时哭，悲喜交并暗催促。
    此理如何举向人，断弦须得鸾胶续。

释宗杲　颂古一百二十一首
    金刚圈，栗棘蓬。
    玄沙三种病，石巩一张弓。
    直截为君说，新罗在海东。

释宗杲　颂古一百二十一首
    杨岐一头驴，只有三只脚。
    潘阆倒骑归，攧杀黄幡绰。

释宗杲　颂古一百二十一首
    老青原，没缝罅。
    问佛法，酬米价。
    差毫釐，成话欛。
    无面目，得人怕。

释宗杲　颂古一百二十一首
    后园驴吃草，一老一不老。
    蓦地撞出来。斩凑得恰好。

释宗杲　颂古一百二十一首
    小鱼吞大鱼，直路太萦纡。
    古帆休更问，处处得逢渠。

释宗杲　颂古一百二十一首
    五逆闻雷，曾能颜回。
    一粒豆子，爆出冷灰。

释宗杲　颂古一百二十一首
    日是祸门，电激雷奔。
    娑竭出海，震动乾坤。

释宗杲　颂古一百二十一首
    日日日日，时时时时，
    违时失候，个老古锥。

释宗杲　颂古一百二十一首
    直出直入，当面不识。
    更拟如何，著甚死急。

释宗杲　颂古一百二十一首
    杀人须是杀人汉，当下一刀成两段。
    头臂夼亏剑刃锋。何似秦时{左车右度}轹鑽。

释宗杲　颂古一百二十一首
    十字待头，现成行货。
    拟欲商量，漆桶蹉过。

释宗杲　颂古一百二十一首
    兔角龟毛眼里栽，铁山当面势崔嵬。
    东西南北无门入，旷劫无明当下灰。

释宗杲　颂古一百二十一首
    云门鉴咦，少有人知。
    咄，无孔铁鎚。

释宗杲　颂古一百二十一首
    一口吸尽西江水，甲乙丙丁庚戊己。
    咄咄咄，囉囉哩。

释宗杲　颂古一百二十一首
    燕坐道场经十劫，一一从头俱漏泄。
    世间多少守株人，掉棒拟打天边月。

释宗杲　颂古一百二十一首
    独坐话谁知，青山对落晖。
    华须连夜发，莫待晓风吹。

释宗杲　颂古一百二十一首
    荷叶团团团似镜，菱角尖尖尖似锥。
    风吹柳絮毛球走，雨打梨花蛱蝶飞。

释宗杲　颂古一百二十一首
    春至自开华，秋来还落叶。
    黄面老瞿昙，休摇三雨舌。

释宗杲　颂古一百二十一首
    壁上安灯盏，堂前置酒台。
    闷来打三盏，何处得愁来。

释宗杲　颂古一百二十一首
    陕府铁牛日癞，嘉州大像耳聩。
    两个病痛一般，咄哉漆桶不快。

释宗杲　颂古一百二十一首
    华阴山前百尺井，中有寒泉彻骨冷。
    谁家女子来照影，不照其馀照斜领。

释宗杲　颂古一百二十一首
    掌内摩尼曾不顾，谁能护惜娘生裦。
