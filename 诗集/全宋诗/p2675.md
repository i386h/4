    毫端真敌万人军，蔬笋能空粉黛群。
    机幕昼閒仍吊古，不胜东合好郎君。

苏泂　金陵杂兴二百首
    半山松菊略能存，犹有荆公旧墨痕。
    佳处不缘谁属我，只应牛首正当门。

苏泂　金陵杂兴二百首
    川原膴膴树斑斑，十进而春风一望间。
    野马不羁天地阔，孤鸿无奈水云间。

苏泂　金陵杂兴二百首
    红袖纷纷转翠屏，元戎小队出郊坰。
    凭谁为集康衢咏，翻作金陵贺太平。

苏泂　金陵杂兴二百首
    阑干三面倚晴晖，竹外长江一线微。
    太守不来僧自看，清凉境界箨龙围。

苏泂　金陵杂兴二百首
    旋结穹庐作露亭，疏枝阑入更轻盈。
    今年客思撩人甚，欲共梅花笑不成。

苏泂　金陵杂兴二百首
    池亭楚楚更堂堂，占得宽閒地一方。
    花竹四时随处有，纵无人赏亦何妨。

苏泂　金陵杂兴二百首
    杏堂缭绕过梅堂，小竹高松映苑墙。
    白雪飞残红雾密，乘舆不到却凄凉。

苏泂　金陵杂兴二百首
    扫地须臾架屋成，风流尤胜拂庐亭。
    吹来一片杯中雪，看著花枝醉忽醒。

苏泂　金陵杂兴二百首
    亭上施屏或可无，江山形胜恐难摹。
    何当穷腊亲来此，身作袁安卧雪图。

苏泂　金陵杂兴二百首
    冶城迤逦过清凉，城脚连云一带长。
    佛寺仙宫更游苑，杖藜知费几诗章。

苏泂　金陵杂兴二百首
    溪上鸳鸯避画船，船中妓女列婵娟。
    使君爱日留连客，同醉江家旧宅边。

苏泂　金陵杂兴二百首
    曲曲清溪有几湾，酒船笳鼓去仍还。
    闲情自合频来此，只在州衙咫尺间。

苏泂　金陵杂兴二百首
    钟山阁上看朝曦，玉作遥山碧四围。
    云气茫茫疑是水，人间无此米元晖。

苏泂　金陵杂兴二百首
    山南山北冢累累，僧寺多承旧宅基。
    兴废相仍真若此，死生常事不须悲。

苏泂　金陵杂兴二百首
    孙陵冈上丛生草，太傅坟前没字碑。
    人世百年元有限，何须功业更巍巍。

苏泂　金陵杂兴二百首
    十里城壕尽是花，覆舟山下几人家。
    日长知道游车倦，自汲新泉唤饮茶。

苏泂　金陵杂兴二百首
    疋马东来说晋元，当时一统笑三分。
    如今寄食城隍庙，异代君臣莫更论。

苏泂　金陵杂兴二百首
    周顒宅作阿兰若，娄约身归窣堵波。
    今日丹青拜荆国，草堂香火已无多。

苏泂　金陵杂兴二百首
    凤凰台上凤凰游，凤去台空水自流。
    惆怅何因见李白，采江试觅月明舟。

苏泂　金陵杂兴二百首
    青山渺渺水迢迢，王气千年久未消。
    独客重来睹风景，江头犹有女吹箫。

苏泂　金陵杂兴二百首
    甫见梅花方结子，即看燕子已生见。
    青春游荡不知返，留住金陵又几时。

苏泂　金陵杂兴二百首
    柳思花情分外娇，青楼消息夜迢迢。
    金陵看得新歌舞，不数吴音共楚腰。

苏泂　金陵杂兴二百首
    竹树池台三亩馀，白云千载护幽居。
    悲凉物色须弹压，粉白脂红却要渠。

苏泂　金陵杂兴二百首
    东冶亭东转半山，栽松庵外万琅玕。
    重来钟阜同僧话，回顾城中几客闲。

苏泂　金陵杂兴二百首
    艳杏夭桃日夜忙，雨馀啼损不成妆。
    元戎和气春风似，又引笙歌看海棠。

苏泂　金陵杂兴二百首
    马军骁骑试楼船，见说倾城出道边。
    元帅留台都赏赐，一时百万喝金钱。

苏泂　金陵杂兴二百首
    纸钱吹挂树枝风，积李崇桃白间红。
    但见牛羊上丘垅，忽看禾黍出蒿蓬。

苏泂　金陵杂兴二百首
    白下纸东送客时，同来何事不同归。
    君归说似西园柳，遮莫轻绵一片飞。

苏泂　金陵杂兴二百首
    北门缭绕出东门，一处亭台酒一巡。
    景物欣欣乐知己，柳容花貌越精神。

苏泂　金陵杂兴二百首
    清明过了日晴佳，九十韶光便有涯。
    恨杀东风太豪横，夜来吹尽海棠花。

苏泂　金陵杂兴二百首
    一派娇波入翠溟，垂杨只似古来青。
    未能草草归家去，怕上劳劳送别亭。

苏泂　金陵杂兴二百首
    城上黄花散漫生，溪头绿树忽阴成。
    浓春自欲烘人醉，却忆秋风一味清。

苏泂　金陵杂兴二百首
    轻风湿湿树斑斑，燕乳鸠鸣春又残。
    门巷狭斜无不到，有何人似柳花閒。

苏泂　金陵杂兴二百首
    张硕风流隐钓缗，渔郎回首漫纷纷。
    杜兰香去知何在，惆怅西江一浦云。

苏泂　金陵杂兴二百首
    扁舟乞得自由身，烂漫金陵作好春。
    桃叶桃根双姊妹，清风明月我三人。

苏泂　金陵杂兴二百首
    九十春光即渐休，莫将双眼上层楼。
    江南归计无由遂，一点杨花万斛愁。

苏泂　金陵杂兴二百首
    月楼谁伴咏黄昏，剪剪轻风阵阵寒。
    独据槁梧成不寐，杏花生影小阑干。

苏泂　金陵杂兴二百首
    不及卢家有莫愁，君恩如水向东流。
    从今相送东流后，地下伤春亦白头。

苏泂　金陵杂兴二百首
    上尽层城更上楼，绿杨枝外尽汀洲。
    蝶衔花蕊蜂衔粉，同向春风各自愁。

苏泂　金陵杂兴二百首
    青帝言归速驾忙，荼醾架底静年芳。
    轻风乍染阴阴绿，微月低笼淡淡香。

苏泂　金陵杂兴二百首
    压架荼醾百万枝，月边花下更涟漪。
    三更比屋皆高枕，如此清凉弃不知。

苏泂　金陵杂兴二百首
    似玉如酥最有情，一香熏彻万愁醒。
    多应无奈花相恼，故绕茶醾架底行。

苏泂　金陵杂兴二百首
    红芳蜕兮花似雾，绿阴径兮叶生烟。
    独往来兮差自得，心境会兮不可言。

苏泂　金陵杂兴二百首
    忆杀仙人白仲都，江边遗迹尚楼居。
    骖鸾翳凤如容我，拜作先师定肯无。

苏泂　金陵杂兴二百首
    紫髯何事忽迁都，有意宫中号太初。
    他日三军嫌饮水，只应翻忆武昌鱼。

苏泂　金陵杂兴二百首
    立夏连朝划地寒，看承芍药倩园官。
    秋千拆罢无聊赖，轩槛周遭绿一般。

苏泂　金陵杂兴二百首
    金城杨柳绿毵毵，摇落殊非昔汉南。
    一语莫因人遂废，树犹如此我何堪。

苏泂　金陵杂兴二百首
    桃花园作芳林苑，蒙汜池通祓禊堂。
    前以骄奢为晋宋，后无规鉴亦齐梁。

苏泂　金陵杂兴二百首
    小虹桥上看朝曦，云里钟山一带微。
    忆在山中望宫阙，淡烟朱碧转依稀。

苏泂　金陵杂兴二百首
    听筝堂上一欷歔，翻手君恩似昔殊。
    按罢金滕应莫怪，近前闲捋使君须。

苏泂　金陵杂兴二百首
    一曲清溪一首诗，风流千载忆僧施。
    小姑溪上平安否，容我骑鱼作伴嬉。

苏泂　金陵杂兴二百首
    方山四面不曾圆，中有仙人号葛玄。
    见说丹成浑不食，五云破洞白升天。

苏泂　金陵杂兴二百首
    不论城外与城中，时节欢然一笑同。
    棠棣花残紫荆老，可无书札问孤鸿。

苏泂　金陵杂兴二百首
    青山绿水思悠悠，见说蛾眉万古愁。
    斗酒百篇能漫浪，白乎吾与女同游。

苏泂　金陵杂兴二百首
    莲花博士去骑鲸，镜曲回瞻一怆情。
    少待诗成附河伯，愿加涂乙始流行。

苏泂　金陵杂兴二百首
    高资冢上五楼穹，先祖曾经相哲宗。
    乱后伤心馀石马，小孙来补万株松。

苏泂　金陵杂兴二百首
    八功德水饮一勺，当下令君热恼消。
    城里红尘莫回首，小屏风岭好逍遥。

苏泂　金陵杂兴二百首
    安榴朵朵簇巾红，数曲阑干掩映中。
    寻得冰纨题小草，浅新还爱绿荷风。

苏泂　金陵杂兴二百首
    前日葵苗匝地青，几何时节与人平。
    旁观指点荼醾架，折蕊攀条似隔生。

苏泂　金陵杂兴二百首
    宛转歌喉一串珠，玉人花貌更清姝。
    明年东武南城唱，试觅源流定忆渠。

苏泂　金陵杂兴二百首
    十八年前蜀道还，维舟曾记此龙湾。
    江流万里元依旧，药市蚕丛兴未阑。

苏泂　金陵杂兴二百首
    旗帜精明鼓角雄，江淮草木尽闻风。
    面门神王须如戟，鼠子无轻怒我公。

苏泂　金陵杂兴二百首
    放散边头武定军，卖刀买犊做农人。
    向来忧国多浮议，谁省官家亿万缗。

苏泂　金陵杂兴二百首
    表里江淮四十州，稍回饥盗事田畴。
    得公数十分天下，庶矣斯民疾痛瘳。

苏泂　金陵杂兴二百首
    三月更当临书头，升元阁畔一闲游。
    偶因花会逢花面，半日僧房得少留。

苏泂　金陵杂兴二百首
    扬州风物转头非，又是葵榴得意时。
    班扇入怀君莫恃，秋来那免弃捐悲。

苏泂　金陵杂兴二百首
    廿载金陵九还往，何如三宿在桑间。
    还家更有江南兴，只取囊诗子细看。

    补得国风成百咏，不妨幕府有诗人。

苏泂　金陵杂兴二百首

苏泂　金陵杂兴二百首

    遍地葱花两三亩，至今人道景阳楼。

苏泂　金陵杂兴二百首

    任是清虚亡晋室，只留卞壶在江东。
