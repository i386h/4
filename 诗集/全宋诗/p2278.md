释普济　偈颂六十五首
    知之一字，众祸之门。
    杀人放火，活捉生擒。
    不是今日枢要，亦非上古典刑。
    与贼过梯了也，攻墙割壁何人。

释普济　偈颂六十五首
    有句无句，如藤倚树，
    截铁斩钉。水不借路，
    树倒藤枯。西天胡子没髭须，
    蹋杀杨歧三脚驴。

释普济　偈颂六十五首
    趁晴盖了屋，乘时刈了禾。
    输纳王租了，鼓腹唱讴歌。

释普济　偈颂六十五首
    天晴日头出，雨落阶头湿。
    宝陀岩上人，犹自不条直。

释普济　偈颂六十五首
    大火已西流，千山爽气浮。
    秋河鸣一雁，砧杵捣离愁。

释普济　偈颂六十五首
    冬到丧后，沙飞石走。
    佛祖潜踪，寸铁在手。

释普济　偈颂六十五首
    圆通门户开，湛然应一切。
    星罗云汉低，山岳起空翠。
    风雨雪霜寒，人惊鸡犬吠。
    正法明如来，依然在门外。

释普济　偈颂六十五首
    一队衲僧来，一队衲僧去。
    打破睦州关，大地无寸土。

释普济　偈颂六十五首
    去日炎威重，归时秋半过。
    林间坐盘石，不觉笑呵呵。
    不笑仰山插锹叉手，不笑地藏博饭栽田。
    二胜米料人人有，笑他饿死饭箩边。

释普济　偈颂六十五首
    祖师不西来，少林有妙诀。
    江南两浙，春寒秋势。
    错！佛法遍天下，
    谈玄口不开。这个是露柱，
    那个是香台。错！
    直饶荐得分明，未免将错就错。

释普济　偈颂六十五首
    禁足护生脚跟下，乞命底纷纷纭纭。
    克期取证平地上，埋没人千千万万。

释普济　偈颂六十五首
    腊极严寒，冻破泥团。
    不可默会，不可形言。
    鸡上树，鸭下水，
    明朝更献楚王看。

释普济　偈颂六十五首
    鹫岭拈花，少林得髓。
    病痛一般，脉息有异。
    脉得著目前见鬼。脉不著死而不吊。

释普济　偈颂六十五首
    栽田博饭，诡计百出。
    插锹叉手，通身荆棘。
    枷棒场中亲不亲，草窠交辊见当人。
    谁是当人，放笋忘恩。

释普济　偈颂六十五首
    击鼓升堂，不谈世法。
    冬至月头，卖被买牛。
    冬至月尾，卖牛买被。

释普济　偈颂六十五首
    经行及坐卧，常在於其中。
    要识其中事，长汀江水东。

释普济　偈颂六十五首
    同条不同条，心眼自相照。
    翻覆论岁寒，不觉拍手笑。

释普济　偈颂六十五首
    绿树阴浓夏日长，楼台倒影入池塘。
    水晶帘动微风起，满架蔷薇一院香。

释普济　偈颂六十五首
    砉碎明月珠，始为家宝。
    颺下鈯斧子，方是住山。
    古策风高，天宽地阔。
    别别，别峰妙转风雷舌。

释普济　偈颂六十五首
    白鹤五通贤圣，瞥喜瞥嗔无定。
    闻名不如见面，见面依然错认。
    从来与渠水米无交，年年今日，
    钵盂安柄。因甚如此，
    鸡不吃谷，肫里有病。

释普济　偈颂六十五首
    西来无意，遍天遍地。
    玉兔东升，金乌西坠。
    老鼠齩床头，猫儿正憨睡。
    莫教动著渠，动著便祸事。
    二祖动著招祸上身，天宁今日开眼失利。

释普济　偈颂六十五首
    风不鸣条，雨不破块。
    老胡忽西来，拨乱清平世。

释普济　偈颂六十五首
    一天星斗大家看，独有渠侬见处偏。
    打失眼晴拈得口，至今脚下墨漫漫。

释普济　偈颂六十五首
    锦城歌管日纷纷，半入江风半入云。
    此曲只应天上有，人间能是几回闻。

释普济　偈颂六十五首
    {假去亻加王}玉已经三日火，探珠曾下九重渊。
    毁珠擿玉无人识，十字街头颺碌砖。

释普济　偈颂六十五首
    山家冬境致，何处不光辉。
    大地冰棱合，漫天雨雪飞。

释普济　偈颂六十五首
    霏霏梅雨洒危层，五月山房冷似冰。
    莫谓乾坤乖大信，未明心地是炎蒸。

释普济　偈颂六十五首
    一新一旧，嘉熙淳佑。
    皇天大历，岁在辛丑。

释普济　偈颂六十五首
    南北山朝对，东西有路分。
    一步不相到，大地入无门。

释普济　偈颂六十五首
    崇山峻岭，茂林修竹。
    终日提纲，神号鬼哭。

释普济　偈颂六十五首
    老黄龙颔下明珠，胜首座悬空扑碎。
    黄檗山吐雾兴云，遍地走南山鳖鼻。

释普济　偈颂六十五首
    为众持钵去，七佛旧仪式。
    庵园塔庙间，处处遍敲击。

释普济　偈颂六十五首
    旱地一声雷，青天撒白雨。
    不是诺佛说底法，不是祖师传底心，
    不是衲僧参底句。日月轮边气象高，
    鱼龙穴下蟠根固。

释普济　偈颂六十五首
    起灭不停，如珠走盘。
    收视返听，坠在黑山。

释普济　偈颂六十五首
    不参禅，不问道。
    竖起铁脊梁，死蛇鑽出划。
    不是禅，不是道。
    云门千岁僧，越山鼐长老。

释普济　偈颂六十五首
    悉达脚下，七花八裂。
    韶阳棒头，有礼有乐。

释普济　偈颂六十五首
    迦叶首传，龙潭只灭。
    一灯一灯，灯灯不绝。

释普济　偈颂六十五首
    南山虎齩大虫，鳖鼻蛇头生角。
    大地六反震动，文殊普贤惊愕。

释普济　偈颂六十五首
    雪压双林树，风号万籁寒。
    瞿昙曾入灭，衲子眼难瞒。

释普济　偈颂六十五首
    铁蒺藜椎，转弄转危。
    丹霞烧木佛，院主落须眉。
    南山炉添今日火，作者不同归。

释普济　偈颂六十五首
    南山云，北山雨。
    毗卢楼阁，豁开门户。
    花簇簇，锦簇簇。
    弥勒大士，倾肠倒腹。
    百城风月，目前领话，
    人畜草芥。一时知有不知有，
    毗目个人执善财手。

释普济　偈颂六十五首
    钟鼓声喧喧，鸟誉闹中庭。
    圆通门不钥，眉毛落又生。

释普济　偈颂六十五首
    冬不寒，腊后看。
    无人知此意，令我忆南泉。

释普济　偈颂六十五首
    明星一见悟无生，攧落阿鼻业火坑。
    三百余番说不出，为他闲事长无明。

释普济　偈颂六十五首
    壁观婆罗门，家业大天下。
    传远枝派分，寸土不存也。
    焚香作忌斋，普请礼三拜。

释普济　偈颂六十五首
    古佛堂前，三头六臂。
    祖师门下，道绝人荒。
    衲僧者里，银山铁壁。
    慧日峰顶，春风和气。

释普济　偈颂六十五首
    验人眼，截流机。
    德山临际，伏惟伏惟。

释普济　偈颂六十五首
    前佛后佛，无生不灭。
    紫磨金色身，衲僧眼中屑。

释普济　偈颂六十五首
    堂中首座人在眼，汉语胡言大藏经。
    七百衲僧吞佛祖，灵山乐得做闲人。

释普济　偈颂六十五首
    觉皇垂迹左金轮，宝殿巍巍表至尊。
    宸翰亲从天上降，烂然金碧耀乾坤。

释普济　偈颂六十五首
    大道无门，十方有路。
    无门不可不入，有路不可不到。
    阿呵呵，参禅，
    不说教。饱饭熟睡游谈，
    三毒无明浩浩。北山免得按牛头吃草。

释普济　偈颂六十五首
    三八念诵，劳动大众。
    五月升堂，说短论长。
    打鼓入室，逼狗透墙。

释普济　偈颂六十五首
    东去西去，南来北来。
    残羹馊饭，门外安排。

释普济　偈颂六十五首
    深山穷谷，蛇虎纵横。
    平田浅划，鸦鸣鹊噪。

释普济　偈颂六十五首
    九十日春，多雨少晴。
    苔痕上阶绿，草色入帘青。

释普济　偈颂六十五首
    地水水风先佛记，冷灰堆里无舍利。
    扫向长江白浪中，千古万古第一义。

释普济　偈颂六十五首
    百尺竿头坐底人，虽然得入未为真。
    百尺午头进一步，十方刹土现全身。

释普济　偈颂六十五首
    鸣爆竹，送残年，
    东村王老夜烧钱。南山短拙无施设，
    借婆裙子拜婆年。

释普济　夹山见船子赞
    毒手一桡难躲避，点头有见问津人。
    烟波浩渺无穷恨，父子虽亲未是亲。

释普济　接待
    目前来往竞纷纷，那个师僧有脑门。
    坐热铁床吞热铁，石头南岳下儿孙　　

释普济　金刚经书大士像赞
    大古应身三十二，一身三十二重非。
    金刚正体是非外，鹊噪鸦鸣无了时。

释普济　进月轩
    桂轮孤朗碧天宽，帘卷清光入座寒。
    底事明明人不荐，　　移花影上栏干。

释普济　觉如周居士圆相赞
    一片虚凝绝谓情，眼横鼻直强图形。
    不如不觉全声色，骂得毗耶老净名。

释普济　兰亭四威仪
    卧，流觞九曲，
    青源白家香熟。三杯席地幕天，
    不辨东西南北。

释普济　兰亭四威仪
    行，崇山峻岭，
    失脚蹋翻古井。通身是口波澜，
    何须三止四请。

释普济　兰亭四威仪
    住，晋朝古寺，
    无米无盐无醋。岁岁遍乞诸邻，
    三百六十余度。

释普济　兰亭四威仪
    坐，茂林修竹，
    高节虚心满目。个个尽是和孙，
    香严口哑舌秃。

释普济　懒瓒赞
    垂颐寒涕满头霜，黄独煨来别有香。
    三诏入云三不起，儿孙各自立封疆。
