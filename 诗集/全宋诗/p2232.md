释法薰　偈倾一百三十三首
    无心亦无法，非法亦非心。
    大勋不竖赏，柴扉草自深。

释法薰　偈倾一百三十三首
    通身无觅处，遍界不曾藏。
    闭目中秋坐，却笑月无光。

释法薰　偈倾一百三十三首
    大地人人具眼，明星夜夜流辉。
    瞿昙不著便，失却一双眉。
    后代儿孙那得知，一年一度乱针锥。

释法薰　偈倾一百三十三首
    院无大小，众无多寡。
    有个折脚铛儿，要人提上挈下。
    不堕功勋，不触风化，
    千里追风看良马。

释法薰　偈倾一百三十三首
    透过牢关，纵横独步。
    脚头到处生涯，谁道此山无路。

释法薰　偈倾一百三十三首
    锺山乍住，百事随宜。
    无可不可，左之右之。
    些子不通方处，料想诸人未知。

释法薰　偈倾一百三十三首
    举古举今，和泥合水。
    说心说性，画虎成狸。

释法薰　偈倾一百三十三首
    石中有玉，沙裹无油。
    德山临济，未出常流。
    却忆寒山子，时临古渡头。

释法薰　偈倾一百三十三首
    四月十五结，虚空遭火热。
    此土与西天，同途不同辙，
    镇萝葡并州铁。

释法薰　偈倾一百三十三首
    今朝五月五，石田无剩语。
    打破鬼门关，三更日卓午，
    明眼衲僧迷却路。

释法薰　偈倾一百三十三首
    深山岩崖中，迥绝无人处。
    个事分明，十成显露。
    拟著便乖张，不拟成差互。
    赵州亲见南泉，百丈曾参马祖。

释法薰　偈倾一百三十三首
    田地稳密，关窍玲珑。
    识得无寒暑处，任他寒暑相攻，
    个裹同途辙不同。

释法薰　偈倾一百三十三首
    长沙见处虽亲用处粗，仰山口裹相呼肚裹狡。
    灼然好个大虫，可惜已伤牙爪。

释法薰　偈倾一百三十三首
    焦砖打著莲底冻，赤眼撞著火柴头。
    将军但得嘉声在，何必荣封万户侯。

释法薰　偈倾一百三十三首
    见闻觉知，行住坐卧。
    眨上眉毛，早已蹉过。
    赤脚唱山歌，路上无人知。

释法薰　偈倾一百三十三首
    直下是，直下是。
    钝鸟不离窠，良驹千万里。

释法薰　偈倾一百三十三首
    身心一如，身外无余。
    长安虽乐，不是久居。

释法薰　偈倾一百三十三首
    禅非意想，道绝功勋。
    青天白日，有迷路人。

释法薰　偈倾一百三十三首
    言诠不及处，文彩未生时。
    电光石火，犹尚迟迟。
    直下横身担荷得，藕丝牵动五须弥。

释法薰　偈倾一百三十三首
    以大圆宽，无绳自缚。
    为我伽蓝，凡寺交参。
    身心安居，换水养鱼。
    平等性智，落在第二。
    莫教辜负两茎眉，九十日从今日始。

释法薰　偈倾一百三十三首
    明如日，黑如漆。
    一犬吠虚，千猱啀宝。
    山僧有口，多时挂壁。
    却烦拄杖子，与诸人通个消息。
    东土衲子心粗，西天蜡人泪滴。

释法薰　偈倾一百三十三首
    今朝端午节，锺山强饶舌。
    家无白泽之图，妖怪自然消灭。
    灭，灭，
    夜行一任贪明月。

释法薰　偈倾一百三十三首
    有祖以来，无人奈何。
    德山临济，特地干戈。
    水到沧溟不是波。

释法薰　偈倾一百三十三首
    未过蒋山关，须过蒋山关。
    过得蒋山关，呼唤不回，
    牢笼不住。平地鼓洪波，
    青天撒白雨。石田笑嘻嘻，
    未敢轻相许。射虎不真，
    徒劳没羽。

释法薰　偈倾一百三十三首
    平如镜，烂如泥。
    迷则迷於悟，悟则悟於迷。
    禹力不到处，河声流向西。

释法薰　偈倾一百三十三首
    江南成橘，江北成枳。
    还有优劣也无，具眼者辨取。

释法薰　偈倾一百三十三首
    慧日峰高，门庭壁立。
    放一线开，从这裹入。

释法薰　偈倾一百三十三首
    一径直，二周遮。
    擒虎兕，辨龙蛇。
    白玉无瑕却有瑕。

释法薰　偈倾一百三十三首
    九重降旨，阖国咸知。
    一句当头，诸人未委。
    只如新长老，宁可截舌，
    不犯国讳。

释法薰　偈倾一百三十三首
    向上一路，千圣不然。
    目前无异草，脚下有青天。

释法薰　偈倾一百三十三首
    众流成海，众毛成球。
    一棚俊鹘，儒雅风流。
    石田无一事，坐地看扬州。

释法薰　偈倾一百三十三首
    石田乍到，人事纷纷。
    拈得鼻孔，失却眼睛。
    佛法二字，不怕烂却，
    且听诸方行棒喝。

释法薰　偈倾一百三十三首
    千五百人善知识，不念吾宗正岑寂。
    五峰趯倒浪翻空，大地山河俱失色。
    金风体露，叶落归根，
    只堪惆怅不堪陈。

释法薰　偈倾一百三十三首
    不结毗卢印，那弘古佛心。
    明月照幽谷，寒涛助夜砧。
    
    
释法薰　偈倾一百三十三首
    至人变化，体妙难名。
    去来不以象，动静不以形。
    通途自寥邈，何处不称尊。
    更无一法可当情，直踏毗卢顶上行。

释法薰　偈倾一百三十三首
    独能岗上，慧日峰前。
    云月是同，溪山各异。
    正法眼，破沙盆。
    劝君更尽一杯酒，西出阳关无故人。

释法薰　偈倾一百三十三首
    风萧萧，雨萧萧，
    丝毫不动，平步丹霄。
    便凭麽去，南山门下，
    土亦难消。

释法薰　偈倾一百三十三首
    赤肉团上，壁立万仞。
    临济德山，因邪打正。
    罗纹结角，百匝千重。
    等闲抛一钓，惊起碧潭龙。

释法薰　偈倾一百三十三首
    寸丝不挂，赤肉犹存。
    黄头碧眼，叫屈无门。
    我行荒草裹，汝又入深村。

释法薰　偈倾一百三十三首
    万古长空，一朝风月。
    有祖以来，话作两橛。
    石田好是明明说，八十翁翁嚼生铁。

释法薰　偈倾一百三十三首
    古今日月，依旧山河。
    新年佛法，特地干戈。
    水到沧溟不是波。

释法薰　偈倾一百三十三首
    尽大地不是自己，参天荆棘。
    尽大地总是自己，满目尘埃横三竖四，
    捏聚放开。要与南山相见，
    还我第一筹来。

释法薰　偈倾一百三十三首
    明如日，黑如漆。
    通天大路，万重铁壁。
    南山口似血盆，争敢妄通消息。

释法薰　偈倾一百三十三首
    至道无难，惟嫌拣择。
    山河大地，日月星辰。
    草木丛林，鸟兽人畜。
    不是龙门客，切忌遭点额。

释法薰　偈倾一百三十三首
    一生空拽露布，末后又把不住。
    灭度不灭度，总非吾弟子，
    破布囊尽底翻出。苍天中更添冤苦，
    可惜黄金成粪土。

释法薰　偈倾一百三十三首
    不著佛求，不著法求，
    不著僧求。春花烂漫，
    春事绸缪。若将春意会，
    特地惹春愁。

释法薰　偈倾一百三十三首
    悉达今朝出母胎，无端满目是尘埃。
    山僧倾倒西湖水，索性从头回洗回。

释法薰　偈倾一百三十三首
    四月十五同结，此土西天不别。
    杀尽安居，只得一橛，
    杨花不是天边雪。

释法薰　偈倾一百三十三首
    是句亦划，非句亦划。
    公案见成，打叠不辨。
    瞎驴趁队过新罗，临济德山空眨眼。

释法薰　偈倾一百三十三首
    眼似流星，机如掣电。
    过得南山门，未见南山面。
    天气稍热。别时相见。

释法薰　偈倾一百三十三首

    退藏於密，得一以宁。
    贵无敌，富无伦。

释法薰　偈倾一百三十三首

释法薰　偈倾一百三十三首

    俊快衲僧，隔墙见角。
    临济德山，蹉过一著。

释法薰　偈倾一百三十三首
    巧尽拙出，水到渠成。
    举足下足，十字纵横。
    三千里外，已被南山换却眼晴。

释法薰　偈倾一百三十三首
    雪峰辊球，禾山打鼓。
    要觅旨归，月不破五。

释法薰　偈倾一百三十三首
    取不得，舍不得，
    状不得。西天碧眼胡，
    尽力道不识。识不识，
    临际德山，太平奸贼。

释法薰　偈倾一百三十三首
    息虑迷源，明机自昧。
    祖师知有处，诸人不知。
    诸人知有处，祖师不会。
    知不知，会不会，
    国有宪章，三千条罪。

释法薰　偈倾一百三十三首
    有祖以来，递相欺诳。
    两两三三，隔靴抓疗山僧有口未曾开，
    早已龟毛长数丈。

释法薰　偈倾一百三十三首
    见得彻，用得亲。
    阳春不同调，入水见长人。
    可惜黄金如粪土，撒向阎沧人不顾。

释法薰　偈倾一百三十三首
    饱丛林，没量汉，
    法战当先，清机历掌。
    假饶千佛出头，也下一槌定当。
