    白日相思可奈何，青春三月已无多。
    桃花风急鲤鱼老，独上台城听踏歌。

苏泂　金陵杂兴二百首
    香罗初换褀衣新，除却垂杨不是春。
    昨日街头骑马过，眼明犹见卖花人。

苏泂　金陵杂兴二百首
    燕子蜂儿各自忙，玉魂谁返杜秋娘。
    人间抵死无殊丽，免唤分司御史狂。

苏泂　金陵杂兴二百首
    睍睆飞来载好音，分明残夏胜春深。
    小窗谁作幽人伴，惟是炉香共绿阴。

苏泂　金陵杂兴二百首
    羊车洗马好谈玄，玉立风姿一少年。
    不枉当时人看杀，如今何处觅幽阡。

苏泂　金陵杂兴二百首
    桂棹青溪夜泊船，胡床歇马坐江边。
    一声笛罢不知处，两不交谈意已传。

苏泂　金陵杂兴二百首
    五马浮江一马龙，亡何兵满建康宫。
    瓜分鼎峙从来事，犹说金陵王气浓。

苏泂　金陵杂兴二百首
    不见乌纱紫绮裘，至今英气满沧洲。
    秦淮半夜棹歌发，明月清天江水流。

苏泂　金陵杂兴二百首
    休论句曲更茅山，只到棲霞说半年。
    暂学僧闲犹未得，几时真个作神仙。

苏泂　金陵杂兴二百首
    三月春归未是归，绿阴端的胜花时。
    绝怜首夏清和句，一字其间几个知。

苏泂　金陵杂兴二百首
    青漆楼高制作奇，当时纯不用琉璃。
    东昏幸自嗤梁武，不道东昏更可嗤。

苏泂　金陵杂兴二百首
    临春何在只桑麻，法宝丛篁间杏花。
    旧日此间同泰寺，曾将龙衮换袈裟。

苏泂　金陵杂兴二百首
    陶山松柏已成阴，地阔天高不可寻。
    二十年中儿又女，人间独自有伤心。

苏泂　金陵杂兴二百首
    寻诗还步小虹桥，薄日轻阴叫百劳。
    芦叶深青荷叶紫，江湖从此兴滔滔。

苏泂　金陵杂兴二百首
    幽草沿阶翠织成，初荷经雨碎珠生。
    藕丝道服松毛扇，独自花间趁鹿行。

苏泂　金陵杂兴二百首
    南轩门第老袁三，将业江湖久遍参。
    闻说中原太平事，数行清泪湿征衫。

苏泂　金陵杂兴二百首
    元帅中军展将旗，军中新刺好男儿。
    风云陈法秋毫识，前日兵兴使不知。

苏泂　金陵杂兴二百首
    缚屋牵牛旋旋归，淮边绿野尽耕犁。
    弓刀武备何曾识，只是当初田舍儿。

苏泂　金陵杂兴二百首
    州衙三面接秦淮，临水朱门一半开。
    却是浙中无此景，江鸥飞舞入城来。

苏泂　金陵杂兴二百首
    潮水萦回御水通，垂杨照影绿茸茸。
    天生此地非疏凿，未必钱塘似此中。

苏泂　金陵杂兴二百首
    一万强人犯海陵，可怜谈笑陷官兵。
    张韩刘岳今何在，塞上将军漫有名。

苏泂　金陵杂兴二百首
    多少横尸叠似楼，官人早晚解红头。
    淮南已是无聊甚，行到江东划地愁。

苏泂　金陵杂兴二百首
    到郡今朝半月日，离家两个许多时。
    知他阿柏无他否，忆我何因我得知。

苏泂　金陵杂兴二百首
    城西二里楚金陵，吴帝名为石首城。
    如今土坞无青草，笑杀当时所必争。

苏泂　金陵杂兴二百首
    观殿今年七十八，往时同饭此斋前。
    梁金华石依然在，屈指重来十五年。

苏泂　金陵杂兴二百首
    七宝装成射雉场，人间还阅几齐梁。
    潘妃自踏金莲去，不见年年春草长。

苏泂　金陵杂兴二百首
    赏心亭前白鹭洲，淮水东畔水东流。
    只知览古生许恨，不知身世自悠悠。

苏泂　金陵杂兴二百首
    青春有酒劝不饮，无乃孤负金陵花。
    主人听客莫怪客，五更酒醒要思家。

苏泂　金陵杂兴二百首
    江鸥野鸭自保社，黑黑白白斗分明。
    东边唼唼西边舞，安得吾身似汝轻。

苏泂　金陵杂兴二百首
    淮南剧贼遽如许，昨日传闻尽杀之。
    罪人斯得勿漫喜，岂有父母残其儿。

苏泂　金陵杂兴二百首
    十分生聚九流亡，鬼面鸦形仰太仓。
    指望淮南大麦熟，夫妻子女得归乡。

苏泂　金陵杂兴二百首
    暄风吹斋响苍玉，竹石相看便有情。
    提胡卢来沽美酒，终日撩人不住声。

苏泂　金陵杂兴二百首
    四十之年又过一，春光回施少年人。
    如丝杨柳看如絮，昼锦堂中又一春。

苏泂　金陵杂兴二百首
    东门草色绿匆匆，游女行寻郎马踪。
    鸡鱼不到吴大帝，签卜争求梁宝公。

苏泂　金陵杂兴二百首
    幕府攒眉为至忧，公堂三五运兵筹。
    归之不得吾何用，独自方桥看水流。

苏泂　金陵杂兴二百首
    雨花台下百花香，驻马坡前百草长。
    若还谷熟人民乐，依旧风流好建康。

苏泂　金陵杂兴二百首
    朱园应是好花开，最忆西庄送别杯。
    火急封书寄子大，少留春色待归来。

苏泂　金陵杂兴二百首
    出门一步便相疏，千里何缘管得渠。
    笑貌声音等如梦，十年参学在长途。

苏泂　金陵杂兴二百首
    短褐钟山不跨驴，杖藜横笛野僧俱。
    虽无字说烦清老，犹恐龙眠画作图。

苏泂　金陵杂兴二百首
    看杀先来所寄书，不知病体比何如。
    想应药裹关心外，更与儿曹作念渠。

苏泂　金陵杂兴二百首
    梦得当时五绝诗，一回吟起一回悲。
    试偷格律无人识，只恐王家燕子知。

苏泂　金陵杂兴二百首
    曲曲青溪贯大江，鸳鸯鸂鶒一双双。
    元戎开燕时来此，城郭山林意两忘。

苏泂　金陵杂兴二百首
    乌衣巷侧长干寺，暇日闲来看一回。
    未觉六朝兴废迹，凄凉先傍眼边来。

苏泂　金陵杂兴二百首
    桑间少妇谁能管，松下孤僧却可怜。
    鱼跃鸢飞均适尔，流行坎止不徒然。

苏泂　金陵杂兴二百首
    钟山窈窕转龙身，小屋低松次第新。
    试觅定林依旧在，只无当日读书人。

苏泂　金陵杂兴二百首
    小盖高肩翼蔽无，钟山寺里换篮舆。
    相逢举止无羞涩，一段风流似上都。

苏泂　金陵杂兴二百首
    魏紫姚黄扫地空，斩新相尚庆元红。
    道人眼界无分别，只赏东来一味风。

苏泂　金陵杂兴二百首
    二十四番风信催，郭南间道有花开。
    园官欲斗金钱赏，名字先供百品来。

苏泂　金陵杂兴二百首
    窗间日日叫提壶，有酒分明不用沽。
    自是老来游兴懒，日长添得睡工夫。

苏泂　金陵杂兴二百首
    细叶如云绿渐敷，玉人晨起换金壶。
    就中一朵紫中贵，除却扬州天下无。

苏泂　金陵杂兴二百首
    四畦芍药间葱花，五月春风便有涯。
    闾巷萧条生意少，隔墙谁信内官家。

苏泂　金陵杂兴二百首
    花案妆来日日新，金陵犹有许多春。
    若为造物回芳意，尽结嘉禾活饿人。

苏泂　金陵杂兴二百首
    始见拖金复散丝，风吹雨打一枝枝。
    灵和殿前好杨柳，争如张绪少年时。

苏泂　金陵杂兴二百首
    琪树三朝种已成，绿阴回合啭宫莺。
    伤心辇路生春草，不见素娥花底行。

苏泂　金陵杂兴二百首
    头上瓶中一万枝，花枝人面两相宜。
    满堂将帅文兼武，都载周家既醉诗。

苏泂　金陵杂兴二百首
    江流带雪少波澜，见说潮头也怕寒。
    未枉推敲瘦诗首，且令澄澈莹中肝。

苏泂　金陵杂兴二百首
    笑谈容易发仓囷，全活生灵百万人。
    十五年来山里坐，不妨来此越精神。

苏泂　金陵杂兴二百首
    老大驱驰亦已劳，移文宁受客嗤嘲。
    人间多少闲田地，无处令渠著把茅。

苏泂　金陵杂兴二百首
    晋宋齐梁几帝陵，土堆谁解有英灵。
    三台五省纷纷改，惟有钟山只麽青。

苏泂　金陵杂兴二百首
    高世何须更欲名，古人于此畅幽情。
    后来同味无王谢，自折幽兰上冶城。

苏泂　金陵杂兴二百首
    安石风流苦未遥，重来遗迹尚山椒。
    一丘元自无花草，底事令人觉意消。

苏泂　金陵杂兴二百首
    车声辘辘马萧萧，山色何如换六朝。
    要识相传风俗在，居人道得语音娇。

苏泂　金陵杂兴二百首
    凤凰台上凤凰游，山自青青水自流。
    千载谪仙元不死，相逢一笑不能休。

苏泂　金陵杂兴二百首
    忆杀当年刘改之，风流曾赋秣陵诗。
    如今白骨埋青草，百咏何因尔得知。

苏泂　金陵杂兴二百首
    平生英俊邢刍父，酷爱闽中一卷诗。
    落笔文章有如此，因何不到秣陵嬉。

苏泂　金陵杂兴二百首
    葛巾裹却葛天民，有女闺中号燕春。
    只在西湖不行脚，金陵风月欠斯人。

苏泂　金陵杂兴二百首
    朱髯五十上南宫，下笔犹能太古风。
    养就胸中治安策，老天那复滞渠侬。

苏泂　金陵杂兴二百首
    中年多病才都减，无奈金陵景物何。
    安得山楹周四者，肯来相伴此吟哦。
