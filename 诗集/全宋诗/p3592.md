    天公念子抄诗苦，借与朝阳小半窗。

杨万里　戏作司花谣呈詹进卿大监郎中
    灵君觞客滕王家，鳌头仙人作司花。
    仙人一笑春风起，开尽仙源万桃李。
    李花冶白桃倡红，坐客桃霞李雪中。
    仙人半酣舞造化，风吹雨打千花空。
    嫣香菸色付一扫，病菡残英又嫌老。
    落花已对春风羞，新花也对春风愁。
    姚黄魏紫世无种，且据眼前桃李休。

杨万里　细雨
    孤闷无言独倚门，梅花细雨欲黄昏。
    可怜檐滴不脱洒，点点何曾离旧痕。

杨万里　峡山寺竹枝词五首
    峡里撑船更不行，棹郎相语改行程。
    却从西岸抛东岸，依旧船头不可撑。

杨万里　峡山寺竹枝词五首
    一水双崖千万萦，有天无地只心惊。
    无人打杀杜鹃子，雨外飞来头上声。

杨万里　峡山寺竹枝词五首
    龟鱼到此总回头，不但龟鱼蟹亦愁。
    底事诗人轻老命，犯滩冲石去韶州。

杨万里　峡山寺竹枝词五首
    一滩过了一滩奔，一石横来一石蹲。
    若怨古来天设险，峡山不到也由君。

杨万里　峡山寺竹枝词五首
    天齐浪自说浯溪，峡与天齐真个齐。
    未必峡山高尔许，看来只恐是天低。

杨万里　峡中得风挂帆
    楼船上水不寸步，两山惨惨愁将暮。
    一声霹雳天欲雨，隔江草树忽起舞。
    风从海南天外来，怒吹峡山山倒开。
    百夫绝叫椎大鼓，一夫飞上千尺桅。
    布帆挂了却袖手，坐看水上鹅毛走。

杨万里　下横山，滩头望金华山四首
    篙师只管信船流，不作前滩水石谋。
    却被惊湍漩三转，倒将船尾作船头。

杨万里　下横山，滩头望金华山四首
    山思江情不负伊，雨姿晴态总成奇。
    闭门觅句非诗法，只是征行自有诗。

杨万里　下横山，滩头望金华山四首
    道是兰溪水较宽，兰溪欲到怪生难。
    後船只羡前船快，不觉前船阁在滩。

杨万里　下横山，滩头望金华山四首
    玉立金华霄汉间，帝封天女鍚仙山。
    被他走下严滩去，一水穷追竟不还。

杨万里　下鸡鸣山诸滩，望柯山不见三首
    地近莲花江渐深，一滩一过快人心。
    贪看下水船如箭，失却柯山无处寻。

杨万里　下鸡鸣山诸滩，望柯山不见三首
    莫怪诸滩水怒号，下滩不似上滩劳。
    长年三老无多巧，稳送惊湍只一篙。

杨万里　下鸡鸣山诸滩，望柯山不见三首
    却忆归舟是去年，上滩浑似上青天。
    诸滩知我怀余怨，急送秋风下水船。

杨万里　下元日诣会庆节所道场，呈余处恭尚书
    琳宫朝谒早追趋，漏尽铜壶杀点初。
    半缕碧云横界月，一规银镜裂成梳。
    自拈沉水祈天寿，散作非烟满王虚。
    已被新寒欺病骨，柳阴偏隔日光疏。

杨万里　夏日绝句
    不但春妍夏亦佳，随缘花草是生涯。
    鹿葱解插纤长柄，金凤仍开最小花。

杨万里　夏日书事
    一字看成两，霜须摘转多。
    既无书册分，奈此日长何？
    荔子红初皱，春醅碧欲波。
    醉来愁自去，不去亦从他。

杨万里　夏日小饮，分题得菱，用辘轳体
    不是齐堂觳觫生，如何双觝独峥嵘。
    犀中忽有红尖角，天上也无甜底冰。
    只爱衣裳桃杏浅，谁知肌骨雪霜明。
    酒徒苦问渠名姓，无姓无名字子陵。

杨万里　夏夜诚斋望月
    山居道是没空庭，不道诚斋敞更明。
    万里青天元是水，半轮皎月忽成冰。
    只今夏热已如此，若到秋高何似生。
    玉兔素娥兼老子，三家一样雪鬅鬿。

杨万里　夏夜露坐二首
    火老殊未热，雨多还自晴。
    暮天无定色，过鸟有归声。
    坐久人将睡，更深月始明。
    素娥欺我老，偏照雪千茎。

杨万里　夏夜露坐二首
    山翠都成黑，天黄忽复青。
    月肥过半璧，云瘦不遮星。
    瓦鼓三四只，村酤一两瓶。
    人皆笑我醉，我独笑渠醒。

杨万里　夏夜耫月
    仰头月在天，照我影在地。
    我行影亦行，我止影亦止，
    不知我与影，为一定为二。
    月能写我影，自写却何似。
    偶然步溪旁，月却在溪里。
    上下两轮月，若个是真底。
    唯复水是天，唯复天是水。

杨万里　夏夜喜雨
    听雨初假寐，还成睡著休。
    梦中帘奏乐，夜半簟知秋。
    今岁应须熟，余生有底愁。
    无人知喜事，课仆织新簄。

杨万里　夏夜月下独酌二首
    谁道今年热，今宵分外清。
    竹风秋九夏，溪月书三更。
    此景天悭与，无人酒自倾。
    明朝火织上，别作一经营。

杨万里　夏夜月下独酌二首
    月忿云遮却，云遭月辊翻。
    斩新灯世界，特地玉乾坤。
    兔舞催诗句，蟾醒笑醉魂。
    姮家且莫睡，照管失金盆。

杨万里　夏夜追凉
    夜热依然午热同，开门小立月明中。
    竹深树密虫鸣处，时在微凉不是风。

杨万里　夏月频雨
    一番暑雨一番凉，真个令人爱日长。
    隔水风来知有意，为吹十里稻花香。

杨万里　夏至後初暑登连天观
    登台长早下台迟，移遍胡床无处移。
    不是清凉罢挥扇，自缘手倦歇些时。

杨万里　夏至雨霁与陈履常暮行溪上二首
    西山已暗隔金钲，犹照东山一抹明。
    片子时间弄山色，乍黄乍紫忽全青。

杨万里　夏至雨霁与陈履常暮行溪上二首
    夕凉恰恰好溪行，暮色催人底急生。
    半路蛙声迎步止，一荧松火隔篱明。

杨万里　鲜鱼潭登舟
    船阁寒沙待晚潮，行人舟子各相招。
    银山一朵三千丈，隔海飞来对面销。

杨万里　闲居初夏午睡起二绝句
    梅子留酸软齿牙，芭蕉分绿与窗纱。
    日长睡起无情思，闲看儿童捉柳花。

杨万里　闲居初夏午睡起二绝句
    松阴一架半弓苔，偶欲看书又嬾开。
    戏掬清泉濂蕉叶，郳童误认雨声来。

杨万里　霰
    雪花遣霰作前锋，势颇张呈欲暗空。
    筛瓦巧寻疏处漏，跳阶误到暖边融。
    寒声带雨山难白，冷气侵人火失红。
    方讶一冬暄较甚，今宵敢叹卧如弓。

杨万里　乡士李英才得老潘墨法，善作墨梅，复喜作诗，艮斋目以三奇，赠之
    吾卿李君磊嵬胸，夜持云梯倚秋空。
    月中夺得修月斧，斫倒南山千岁松。
    束归丹灶和玉桂，燧出绿雾霏鸾龙。
    捣成玄圭与苍璧，洒作横枝岁寒色。
    庾岭霜林和靖园，掇人生绡供戏剧。
    幻松作璧璧作梅，豪气勃郁尚不开。
    琼艘满釂梅雪下，吐出西湖有声画。

杨万里　芗林五十咏　白鹤亭
    可怜林下鹤，独肯伴先生。
    犹有冲天志，君听叫月声。

杨万里　芗林五十咏　百花洲
    茂苑犹香名，南阳已荒皁。
    更兼芗林中，三洲是三岛。

杨万里　芗林五十咏　北垞
    住处何曾远，林间别是凉。
    清风酣一枕，底许觅羲皇。

杨万里　芗林五十咏　碧芦淙
    春笋肥堪菜，秋花暖可毡。
    何须双野鸭，屏上五湖天。

杨万里　芗林五十咏　碧梧
    往往游西崦，时时憩午阴。
    凤来人不见，飞入碧枝深。

杨万里　芗林五十咏　丛桂
    不是人间种，移从月胁来。
    广寒香一点，吹得满山开。

杨万里　芗林五十咏　东皋
    避世无功酒，临流元亮诗。
    鹤鸣入天听，叶下报秋悲。

杨万里　芗林五十咏　读书堂
    老以书为命，眼将书作雠。
    芗林与诚叟，同贩短檠愁。

杨万里　芗林五十咏　鹅篆滩
    不傅右军字，且留房相池。
    新雏黄似酒，把酒看群嬉。

杨万里　芗林五十咏　泛宅
    犹道山中浅，仍移水上居。
    俗人又剥啄，掉入白芙蕖。

杨万里　芗林五十咏　放鱼矶
    为惜卅砂尾，亲投碧草潭。
    前头有香饵，千万不须贪。

杨万里　芗林五十咏　芙蓉沜
    岸植木菡萏，池栽水拒霜。
    那知一家种，同艳不同香。

杨万里　芗林五十咏　柑子园
    秋颗酣天酒，春花噀国香。
    芗林亲手种，中有洞庭霜。

杨万里　芗林五十咏　瓜田
    独酌圣贤酒，新尝子母瓜。
    丁宁林下友，莫道故侯家。

杨万里　芗林五十咏　皈来桥
    已赓彭泽辞，更拟辋川诗。
    未老还山了，犹嫌皈较迟。

杨万里　芗林五十咏　过花桥
    北垞通南埭，前花映後花。
    雨多桥板断，折碎两涯霞。

杨万里　芗林五十咏　海棠洞
    艳翠春销骨，妖红醉入肌。
    花仙别无诀，一味服燕支。

杨万里　芗林五十咏　黑坳
    坳中一池墨，腕中百斛力。
    洒作五十诗，千年照泉石。

杨万里　芗林五十咏　槐阴墿
    阴作官街绿，花催举子黄。
    公家有三树，犹带凤池香。

杨万里　芗林五十咏　驾月桥
    紫霓横天度，银盘涌水来。
    先生骑霓脊，盘上卷金杯。

杨万里　芗林五十咏　菊坡
    西风破幽馥，东篱散清好。
    渊明不可作，解后芗林老。

杨万里　芗林五十咏　兰畹
    健碧缤缤叶，斑红浅浅芳。
    幽香空自秘，风肯秘幽香。

杨万里　芗林五十咏　梨花原
    禁火晓未暖，蹈青昏恰皈。
    何堪一原雪，将冷入春衣。

杨万里　芗林五十咏　连辉观
    极目斜阳树，乘风落月梁。
    更和山与水，并作一秋光。

杨万里　芗林五十咏　梅坡
    自昔花如雪，而今叶入云。
    先生已仙去，谁与共天醇。

杨万里　芗林五十咏　幕翠庭
