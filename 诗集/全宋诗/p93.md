曹勋　山居杂诗九十首
    春雨要如膏，发生物已拆。
    霡霂则易耨，浸渍乃损麦。
    连旬不见日，过社麻亦阨。
    再拜祈开晴，民食庶可得。

曹勋　山居杂诗九十首
    新年供家庙，老稚晨夕忙。
    块坐温暖处，但闻油酱香。
    不问物物阙，有食时时尝。
    诸孙甚解事，翁乃无饥肠。

曹勋　山居杂诗九十首
    蓬荜久不居，乍归欣故情。
    落叶聚蛛緒，残莎闹蛩声。
    迹与朝市远，梦随风月清。
    从此眠食间，萧散安可得

曹勋　山居杂诗九十首
    林泉久自得，窗户向深幽。
    不复嫌村酒，何能有客愁。
    夕阳烘远水，飞絮挟轻鸥。
    杖屦归从晚，柴门月一钩。
    
    
曹勋　山居杂诗九十首
    八月河汉近，凉蟾辉太清。
    窗户纱已薄，草木风有声。
    所喜暑云退，不惮岁徂征。
    招客对晚色，酒与月同倾。

曹勋　山居杂诗九十首
    中春艳桃李，东风惠以繁。
    胡为连日雪，花叶飘阑干。
    气非骄阳候，宛若残腊寒。
    玄冥僭融和，玉川知其端。


曹勋　山居杂诗九十首
    村落辟幽居，深静颇得趣。
    白云栖檐栋，岚光照窗户。
    往还得隐逸，谈笑皆旧故。
    不畏归路遥，家家响机杼。

曹勋　山居杂诗九十首
    新春遨西湖，结伴每有侣。
    岂意气差忒，旬中自寒暑。
    欣欣一日晴，愀愀三日雨。
    谓言爱酒伴，有酒谁能湑。

曹勋　山居杂诗九十首
    手种亭下松，偃盖出云外。
    攀条喜修干，为我发清吹。
    晓庭风瑟缩，夜月金锁碎。
    种久方来归，对之颜有愧。

曹勋　山居杂诗九十首
    寓居溪阴刹，未暇理归棹。
    遮眼阙文史，借书得探讨。
    卷舒白虎通，涵咏青龙钞。
    贤哉绝世书，欣然慰幽抱。

曹勋　山居杂诗九十首
    门前露草嫩，极目遨青龙。
    策杖不知远，看山兼别峰。
    晓厨寻晚笋，客供摘新菘。
    舌下清肥足，何劳细肋供。

曹勋　山居杂诗九十首
    我有旧知识，生来晦踪迹。
    邂逅不可见，早夜何能觅。
    忽然积羽处，相逢吐胸臆。
    临了一句子，会心极得力。

曹勋　山居杂诗九十首
    吾年已七十，眠早起不宴。
    倦甚就宵枕，卧久思永旦。
    虽无文字劳，渐绝人事叹。
    收其放心尔，戒得尤所赞。

曹勋　山居杂诗九十首
    得闲一何乐，佳处先徜徉。
    每扣隐逸士，屡过清静坊。
    地僻干戈远，年丰禾稻香。
    访我亦何所，南涧与东冈。

曹勋　山居杂诗九十首
    斗水小涟漪，拳石献岩窦。
    苍苍三峰足，挺挺数叶秀。
    棐几得巨壑，幽色清永昼。
    以我兼忘心，嵩华亦何有。

曹勋　山居杂诗九十首
    架屋集匠氏，斲石最可恤。
    相石千仞端，壍山方得出。
    疲形困辇置，巧磓较丝忽。
    所直等众人，人可不择术。

曹勋　山居杂诗九十首
    秋雨湿庭户，陋舍集闲客。
    重以樽酒空，可为饥肠厄。
    呼童取杯酌，村酒我先识。
    一杯复一杯，满坐生春色。

曹勋　山居杂诗九十首
    秋雨不可雷，雷即雨无节。
    况当收刈时，只欲天明洁。
    夫何连日雨，卧穗恐芽蘖。
    已失早禾望，晚田幸天辍。

曹勋　山居杂诗九十首
    陈梨甘若饴，雪梨脆而水。
    二物浙左右，大颗擅甘脆。
    秋热兼宿酒，啗之手不置。
    虽非凤浊鹅，惠予亦多矣。

曹勋　山居杂诗九十首
    山色侵衣冷，异气作心赏。
    岭月闻凤笙，松风吹鹤氅。
    碧池绉澄绿，群山发清响。
    以此助萧闲，欣然同击壤。

曹勋　山居杂诗九十首
    台蕈甘擅名，油葱芼枯腊。
    及居台山下，生致出岩隙。
    丁头形磷磷，鸡股香羃羃。
    清对云水客，荤素得共食。

曹勋　山居杂诗九十首
    休退居远邑，顿喜人事疏。
    饭箩自不足，日记无可书。
    惟对楮先生，复接松大夫。
    溪友留白鱼，鼾睡惊长须。

曹勋　山居杂诗九十首
    寂历斜阳岸，高低犹涨痕。
    轻舟随浪稳，远水带烟昏。
    野色几番换，乔林何处村。
    情怀了无事，安得易销魂。

曹勋　山居杂诗九十首
    晨粥山中行，落叶响步武。
    委曲访隐逸，清谈销百虑。
    老汤瀹苦茗，宿火煨山芋。
    却照夕阳归，家人不知处。

曹勋　山居杂诗九十首
    素原游匡庐，梦寐不可易。
    弃官竟一往，不以归意近。
    屡扣清虚仙，杯酒话畴昔。
    永怀拨云峰，西麈每相忆。

曹勋　山居杂诗九十首
    初意得闲适，处处期登临。
    岂谓病见拘，未能摅灵衿。
    小滞广莫原，恐辜云水心。
    强半人事尔，无令成侵寻。

曹勋　山居杂诗九十首
    饭了纵闲适，杖屦涉晴昼。
    时物破百感，环山献明秀。
    承宇茂芳桂，追凉风细逗。
    摘果靡惮劳，聊用引稚幼。

曹勋　山居杂诗九十首
    临老看草圣，作字意少快。
    居然落笔际，点画反留碍。
    早意亦易事，怀抱殊不介。
    兹来每把玩，徒复一叹慨。

曹勋　山居杂诗九十首
    前日花正红，今旦香已入。
    我知色空相，未开固非在。
    韶华每备陈，风雨不本贷。
    任渠万点飞，岂待愁如海。

曹勋　山居杂诗九十首
    岁华念将晚，寒色惨中抱。
    昔年困虀盐，拨灰吟苦调。
    老来适其适，所乐在燕笑。
    招宾共一醉，围炉不妨小。

曹勋　山居杂诗九十首
    吾家远城市，四围萦叠嶂。
    意寻烟霞游，喜逐樵牧上。
    松间憩两足，论高等四畅。
    或时戴月归，山花记心想。

曹勋　山居杂诗九十首
    北内号枯桑，浓寒坚夜冻。
    关河鸟过少，雪意晓来重。
    陌上未沾尘，山中已成垅。
    雝雝南去雁，袖手徒目送。

曹勋　山居杂诗九十首
    雅意老山林，每作山林趣。
    引领山林景，赋咏山林句。
    岚光增日夕，林风度朝暮。
    时时携筇归，群山送归户。

曹勋　山塞曲
    来时长驱五千里，满目胡尘涨天起。
    胡尘已息辽海空，班师来谒明光宫。
    汉家天子耀神武，不知战士常辛苦。

曹勋　山中
    告才吾君得挂冠，交亲犹肯问平安。
    连天秋色雁已过，满槛竹声风逗寒。
    对客杯盘心共醉，催人节物岁将阑。
    但知适意山林去，不觉山头行路难。

曹勋　山中二首
    山中从事亦贤劳，安得眠云独自高。
    两席地栽青紫芥，一株树接碧红桃。
    春风多可移花性，夜雨无声入土膏。
    那办工夫到诗句，溪灯穿屋未编茅。

曹勋　山中二首
    人间久矣倦迎逢，归路牛羊带夕舂。
    千点梅沉山店月，一溪烟咽寺楼钟。
    穷犹羞涩巉岩面，老自平夷磊磈胸。
    曾笑古人多晚谬，草庵虽小幸相容。

曹勋　山中谣
    山中之乐何由说，知者不言言者拙。
    红尘飞尽散松风，独酌寒泉弄明月。

曹勋　扇车
    良工巧制鲍鱼形，短架圆机扇比名。
    十幅怀箱尘不到，一轮因刀羽犹轻。
    辗回剧暑轩楹去，载得长风枕簟清。
    失喜门边须茗客，隔篱悮作碾时声。

曹勋　上塘值清明八首
    客路逢寒食，划船闹水干。
    娇云易得雨，暴冷不成寒。
    白水连天去，繁桃触目看。
    朝来有家信，稚子报平安。

曹勋　上塘值清明八首
    客路逢寒食，清明未到家。
    起居须药饵，饮食任鱼暇。
    水净垂垂柳，风迟艳艳花。
    兴来随处乐，不复问年华。

曹勋　上塘值清明八首
    客路逢寒食，犹寻御腊衣。
    群芳须暖景，曨日尚寒晖。
    余力求闲早，劳生昨梦非。
    荠花浑似雪，蝴蝶梦中归。

曹勋　上塘值清明八首
    客路逢寒食，光阴亦可惊。
    如何介子节，再过阖庐城。
    列岫遥逾碧，群芳晚更明。
    饭余风力壮，掀舞小舟轻。

曹勋　上塘值清明八首
    客路逢寒食，山光触处新。
    取鱼多样纲，舣岸几州人。
    默尔非无住，超然得任真。
    今年卦气尽，四顾欲何亲。

曹勋　上塘值清明八首
    客路逢寒食，浙西春事稀。
    日华藏暖景，天气未单衣。
    歌棹喧椎鼓，桃花巧映扉。
    空蔬黣麦粉，药病岂心违。

曹勋　上塘值清明八首
    客路逢寒食，禾身出已齐。
    清溪三汊浅，白鹭一行低。
    禊饮心何在，茅峰路恐迷。
    随时随有乐，何处不幽栖。

曹勋　上塘值清明八首
    客路逢寒食，欣同不系舟。
    闲中聊适性，要地固抽头。
    不作扬州梦，何能贾客留。
    壮怀今断断，世事任悠悠。

曹勋　上元前四日宣召赏梅
    冉冉东风春乍回，睿思宣入禁中来。
    忽参鼻观清香彻，殊喜一庭都是梅。

曹勋　上云乐
    真王严仗卫，清道表前旌。
    翠辇浮龙藻，霜戈丽虎兵。
    哀笳鸣广路，警跸过曾城。
    后乘超王屋，前驱上玉京。
    诸天观鼓节，列圣肃仪刑。
    朝谒虚皇罢，麾幢下始青。
    蒷兵宁紫户，阅籍按黄庭。
    十洲皆承事，九宇仰裁成。
