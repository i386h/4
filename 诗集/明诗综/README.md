　　提要
　　【臣】等谨案明诗综一百巻
　　国朝朱彞尊编彞尊有经义考已着録明之诗派始终三变洪武开国之初人心浑朴一洗元季之绮靡作者各抒所长无门户异同之见永乐以迄化治沿三杨台阁之体务以舂容和雅歌咏太平其也冗沓肤廓万喙一音形模徒具兴象不存是以正徳嘉靖隆庆之间李梦阳何景明等崛起于前李攀龙王世贞等奋发于后以复古之説递相唱和导天下无读唐以后书天下向应文体一新七子之名遂竟夺长沙之坛坫渐久而摹拟剽窃百俱生厌故趋新别开蹊径万厯以后公安倡防诡之音竟陵标幽冷之趣么侧调嘈囋争鸣佻巧荡乎人心哀思闗乎国运而明社亦于是乎屋矣大抵二百七十年中主盟者递相盛衰偏祖者互相左右诸家选本亦遂皆坚持畛域各尊所闻至钱谦益列朝诗集出以记丑言伪之才济以党同伐异之见逞其恩怨颠倒是非黒白混淆无复公论彞尊因众情之弗协乃编纂此书以纠其谬每人皆畧叙始末不横牵他事巧肆讥弹里贯之下各备载诸家评论而以所作静志居诗话附于后虽隆万以后所收未免稍繁然世逺者篇章易佚时近者部帙多存当亦随所见闻不尽出于标榜其所评品亦颇持平于旧人私憎私爱之谈往往多所匡正六七十年以来谦益之书巳澌灭无遗而彞尊此编独为诗家所传诵亦人心彞秉之公有不知其然而然者矣乾隆四十六年四月恭校上

　　太祖高皇帝【三首】
　　帝讳元璋姓朱氏字国瑞濠之钟离东乡人元至正十一年辛夘起兵丁未称吴元年戊申建元洪武在位三十一年崩孝陵【在应天府治东北钟山之阳】永乐元年上尊諡曰圣神文武钦明启运俊徳成功统天大孝髙皇帝庙号太祖嘉靖十七年改上尊諡曰开天行道肇纪立极大圣至神仁文义武俊徳成功髙皇帝有御制诗集五巻
　　神凤操
　　钧天奏兮列丹墀俄翩翩兮凤皇仪敛翺翔兮栖梧枝彼观徳兮直为我辞
　　钟山赓吴沉韵
　　嵯峨倚空碧环山皆拱伏遥岑如剑防迩洞非茅屋青松秀紫崖白石生谷岩畔毓灵芝峯顶森神木时时风雨生日日山林沐和鸣尽啼莺善举皆飞鹄山中道者禅陇头童子牧试问防经年答云常辟谷
　　春望牛首
　　遥岑峙立势苍然春日莺啼景物鲜叠嶂倚天江月外三山映带石城邉
　　【刘基曰皇帝提一旅之众龙飞淮甸不十年间奄有区宇自古以来武功之盛未之有也及夫万防之暇作为文章举笔立就莫不雄深宏伟言雅而防逺真所谓天生聪明可望而不可及者矣　宋濂曰皇帝临御以来用人文化成天下天纵圣能形诸篇翰挥洒之际不待凝神而思若渊泉有长江大河一泻万里之势自然度越千古　郭传曰自汉而下兴王代作然文经武纬一弛一张率不能兼济其美惟唐太宗以兵力定乱及践祚之后词章奋发论者以其雕竒镂彩徒与骚人韵士争锱铢之巧惟皇上以一旅取天下不数年间混一南北至万防之暇复游心道学志存诗书寓辞竹帛亲搦管翰意无停机一挥数千百言粹然出于情性之正觧缙曰臣少侍髙皇帝早暮载笔墨楮以俟圣情尤善为诗歌睿思英发雷轰电烛玉音沛然数千百言一息无滞臣辄草书连幅笔不及成防画上进才防定数韵而巳或不更一字故尝喜诵古人铿鍧炳朗之作尤恶寒酸吚嘤龊龌鄙陋以为衰世之调为不足观诗僧宗泐进所精思刻苦以为得意之作百余篇上一览不竟日尽和其韵雄深阔伟下视泐诗真大明之于爝火也　陆釴曰洪惟太祖应天顺人创业垂统巍乎成功焕乎文章称盛羙矣　郑哓曰上初不识书每退朝暇延接儒士讲论经典又取古帝王嘉言善行书之殿庑出入省视作为文章挥笔立就或命侍臣立榻下操觚授词混混千言皆淳雅髙简洞逹物情动引经史足法万世　髙岱曰太祖以武功定天下而崇尚文学如饥渴之于饮食每得儒臣皆待以腹心帷幄朝夕谘访不倦至于解析经义又多天纵神啓有非老师宿儒之所能及者　朱国祚曰髙皇帝诗发乎天籁自然成音当日赓歌飏言匪仅詹吴乐宋四学士而巳若逹鲁与权吴伯宗刘仲质张翼宋璲朱芾桂慎戴安王厘周衡吴喆马从马懿易卢均裴植李睿韩文辉曹文夀单仲右诸臣进诗咸赓其韵今诸臣之作十九无闻而圣藻长垂宇宙髙矣羙矣静志居诗话孝陵不以马上治天下云雨贤才天地大文形诸篇翰七年而御制成集八年而正韵成书题诗不惹之庵置酒滕王之阁赏心胡闰苍龙之咏击节王佐黄马之謡日歴成编和黄秀才有作大官设宴醉宋学士有歌顾天禄经进诗篇披之便殿桂彦良临池聫句嫓于飏言韵事特多更仆难数惟其爱才不及因之触物成章宜其开创之初遂见文明之治江左则髙杨张徐中朝则詹吴乐宋五先生蜚声岭表十才子奋起闽中而三百年诗教之盛遂超轶前代矣】
　　成祖文皇帝【一首】
　　帝讳棣髙皇帝第四子初封燕王之国北平建文元年称兵靖难四年入金川门建文帝自焚【或传出亡】帝即位革除建文年号称洪武三十五年逾年建元永乐在位二十二年崩于榆木川长陵【在昌平州北一十八里天夀山】諡曰体天道髙明广【嘉靖十七年更广为肇】运圣武神功纯仁至孝文皇帝庙号太宗嘉靖中更成祖有御制集
　　勃泥长宁镇国山诗
　　【实録永乐六年十二月丁丑遣中官张谦行人周航防送嗣浡泥国王遐旺等还国初故浡泥国王麻那惹加耶乃言蒙朝廷厚恩赐封王爵国之境上皆属职方而国有后山乞封为一国之镇至是其子遐旺复以为请遂封其山为长宁镇国之山命谦等即其地树碑上亲制文系之以诗】
　　炎海之墟浡泥所处喣仁渐义有顺无忤慺慺贤王惟化之慕导以象胥遹来奔赴同其妇子兄弟陪臣稽颡阙下有言是陈谓君犹天遣其休乐一视同仁匪偏厚薄顾兹鲜徳弗称所云浪舶风樯实劳恳勤稽古逺臣顺来怒趌以躬或难矧曰家室王心亶诚金石其坚西南蕃长畴与王贤矗矗髙山以镇王国镵文于石懋昭王徳王徳克昭王国休宁于万斯年仰我大明
　　【沈徳符曰文皇帝封外国山者凡四永乐三年十月封满刺加国西山曰镇国山四年正月封日本国镇山曰夀安镇国之山六月十二月封浡泥后山曰长宁镇国之山十四年十二月封柯枝国中山曰鎭国山皆出御制诗文以炳燿海外且词旨隽蔚断非视草解杨诸臣所能办因思唐文皇兵力仅伸于漠北而屈于辽水若文皇帝威徳直被东南古所未賔之国赑屭穹碑昭回云汉万禩所无也】
　　【诗话长陵称兵靖难践位之后加意人文成均视学有碑阙里褒崇有述以及姑苏寳山之石武当太岳之宫靡不亲制宸章勒之丰碣而又五经四书性理大全有书圣学心法有书大典有书文华宝鉴有书为善隂骘有书孝顺事实有书务本之训有书不独纪之以事抑且系之以诗至于过兖州则赐鲁藩于吴则夀荣国交阯明经甘润祖等一十一人庶常吉士曽棨等二十八宿咸承宠渥三勒石于幕南之廷四建碑于海外之国此諡法之所以定为文与由是文子文孙复加继述内阁则有三杨翰林则有四王尚书则东王西王祭酒则南陈北李观灯侍宴拜手赓歌呜呼盛矣】
　　仁宗昭皇帝【二首】
　　帝讳髙炽成祖长子建元洪熈在位一年崩献陵【在天夀山西峯之下】諡曰敬天体道纯诚至徳文钦武章圣逹孝昭皇帝庙号仁宗有御制文集二十巻诗集二巻
　　冬至赐赞善徐好古
　　清朝盛文治辅徳资儒耆念彼筋力倦趋朝谅非宜赋诗有佳致纳诲多良规起予徳深趣欢懐浩无涯新阳届初复况此承平时酬劳有尊酒庶以劳期颐
　　池亭纳凉
　　夏日多炎热临池憩午凉雨滋槐叶翠风过藕花香舞燕来青流莺出建章援琴弹雅操民物乐时康【诗话献陵天禀纯明雅志经术东朝监国命徐赞善善述纂尚书直指进讲诗成亦命善述改窜令防呼其字而不名又尝与曽少詹棨赓和如江楼秋望诗云苹洲晴亦雪枫岸昼常霞絶似唐太宗设享年加永则成功文章巍焕何如焉】
　　宣宗章皇帝【九首】
　　帝讳瞻基仁宗长子建元宣徳在位十年崩景陵【在天夀山东峯之下】諡曰宪天崇道英明神圣钦文昭武寛仁纯孝章皇帝庙号宣宗有御制文集四十四巻诗集六巻
　　猗兰操
　　昔孔子自卫反鲁隠居国中见兰之茂与众草伍自伤不逢时而托为此操予虑在野之贤有未出者故拟作焉
　　兰生中谷兮晔晔其芳贤人在野兮其道则光嗟兰之茂兮与众草为伍呜呼贤人兮汝其予辅
　　思贤诗【有序　宣徳九年正月】
　　予嗣守祖宗大位夙夜兢惕思惟致治之道必有贤臣相与赞辅虽屡诏求贤然恭黙之思未已乃作诗以着予志诗曰
　　天命有赫付畀万方肆予承之夙夜弗遑亮天之工其责在予亦惟求贤以作心膂尧舜大圣咨于臣邻汤武致治敷求哲人稷契臯防周召伊傅同徳一心以匡以辅惟时匡辅百工允厘治效之隆臻于皞熙悠悠我心念之勿置欲得羣贤以弼予治告言惓惓束帛戋戋命彼皇华歴于丘园庶防多才防茅连茹奋其功庸翼我王度惟天昭昭维岳降灵笃生贤哲聿驰骏声啓心沃朕以迪先徳扬其耿光有永无斁
　　赐许廓廵抚河南
　　【实録许廓字文超河南襄城人由国子生授锦衣卫经歴擢工科给事中转鸿胪左寺丞升兵部右侍郎洪熙元年转左侍郎宣徳六年升兵部尚书尝扈从北征董饟运交阯入軄方朝廷遣廓徃覆实戸口田赋寻召还后命招抚河南流民归者数万戸尚书张本卒以廓代之】
　　河南百州县七郡所分治前嵗农事缺旱潦始复继衣食旣无资民生曷由遂顾予位民上日夕懐忧媿尔有敦厚资其徃勤抚字徙者必辑绥饥者必赈济咨询必周歴毋惮躬劳勚虚文徒琐碎所至皆实恵勉旃罄乃诚庶用副予意
　　悯农诗示吏部尚书郭璡
　　【雷礼列卿纪郭璡初名进字时用保定新安人由国子生出身永乐初为戸部主事歴迁福建参议山东参政进工部右侍郎改吏部洪熈初兼少詹事宣徳初命掌行在詹事府事进尚书】
　　农者国所重八政之本源辛苦事耕作忧劳亘晨昬丰年仅能给歉嵗安可论旣无穅覈肥安得缊絮温恭惟祖宗法周悉今具存遐迩同一视覆育如乾坤尝闻古循吏卓有父母恩惟当慎所择庶用安黎元
　　示户部尚书夏原吉
　　闗中嵗屡歉民仓无所资郡县旣上言能不轸恤之周礼十二政散货首所宜给帛遣使者发廪饬有司临轩戒将命遄徃毋迟迟命下茍或后施济安所期吾闻有道世民免寒与饥循巳不遑宁因请书媿词
　　捕蝗诗示尚书郭敦
　　【雷礼列卿纪郭敦字仲厚东昌棠邑人洪武癸酉中乡试后入太学擢戸部主事出知衢州府庶务毕举召为监察御史升河南布政司左参政调陜西以礼部尚书胡濙荐擢礼部右侍郎廵抚顺天等处洪熈初降太仆卿旋进戸部左侍郎兼詹事府少詹事宣徳二年升戸部尚书】
